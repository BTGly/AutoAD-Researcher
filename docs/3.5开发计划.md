# 0. 阶段定义

Step 3.5 接收 3.4 产出的用户选择的 1–3 个实现 Variant，为其生成**共享评价协议下的统一实验计划包**。

核心不变量：

```
一个 run → 一个 SharedExperimentProtocol（baseline/meta/evaluator/reuse policy）
共享协议 → 显式 seed 列表 → baseline 与所有 Variant 完全相同
共享协议 → StatisticalAnalysisPlan（运行前固定，禁止运行后修改成功标准）
共享协议 → N 个 VariantTrialSpec（N ∈ [1, 3]）
每个 Variant → fit(可选) + smoke + full 三个 TrialIntent
所有实验单元 → ExperimentMatrix（baseline × seeds + variant × fit/smoke/full + variant × seeds）
依赖关系 → 只在 Matrix 中表达
资源估算 → 只在 resource_budget.json 中表达，Matrix 不含资源数字
```

## 0.1 职责边界

3.5 做：
- 从 3.4 handoff 消费已确认 Idea 和用户选择的 Variant
- 设计共享实验协议（PlanningInputRefs + 评价链覆盖验证）
- 定义 StatisticalAnalysisPlan（ScientificConclusion enum + 确定性规则 + 完全覆盖）
- 为每个 Variant 生成试验规格（baseline/fit/smoke/full intents）
- 生成 ExperimentMatrix（baseline × seeds + variant × fit/smoke/full + variant × seeds）
- 将 3.4 unresolved dimensions 编译为 ExperimentalResolutionPlans（Matrix 生成后，绑定 entry_ids）
- 汇总 ResourceBudget + BudgetDecision（revision 生命周期闭环，protocol 版本化）
- 定义 OperationalGuardPolicy（按 target_entry_ids 绑定，不含科研提前终止）
- 10 类不变量确定性预执行验证
- 产出 SHA-only 引用的 Handoff

3.5 不做：
- 自行读取仓库文件（Permission Profile deny repository_filesystem_read）
- 重新读取论文 PDF（3.2 已做）
- 重新扫描仓库源码（3.1 已做）
- 生成文件级 PatchPlan（3.6）
- 生成具体 shell 命令或脚本文件名（3.6+3.7）
- 执行代码或运行实验（3.7）
- 替用户静默确认任何决策
- 运行期间根据已观测指标提前终止 seed（optional stopping）

from autoad_researcher.schemas.transfer_design import (

class Stage34HandoffError(Exception):
    """3.4 handoff 结构不满足 3.5 要求时抛出。"""

class Stage34InputAdapter:
    """将 3.4 正式 handoff 翻译为 3.5 内部输入。

    每个 selected variant 必须存在对应的 VariantTransferAnalysis 和 VariantRiskReport。
    缺失 → Stage34HandoffError，不静默继续。
    """

    def load(self, handoff: IdeaTransferDesignHandoff) -> Stage35Input:
        analyses = handoff.transfer_analysis.variant_analyses
        risk_by_id: dict[str, VariantRiskReport] = {}
        for r in handoff.variant_risk_reports:
            if r.variant_id in risk_by_id:
                raise Stage34HandoffError(
                    f"duplicate VariantRiskReport.variant_id: {r.variant_id}"
                )
            risk_by_id[r.variant_id] = r

        if not handoff.selected_variants:
            raise Stage34HandoffError("selected_variants must not be empty")

        variant_inputs = []
        for v in handoff.selected_variants:
            analysis = analyses.get(v.variant_id)
            if analysis is None:
                raise Stage34HandoffError(
                    f"selected variant {v.variant_id} has no VariantTransferAnalysis"
                )
            if analysis.variant_id != v.variant_id:
                raise Stage34HandoffError(
                    f"analysis.variant_id {analysis.variant_id} != variant {v.variant_id}"
                )
            if analysis.overall_status in {"non_viable", "needs_reanalysis"}:
                raise Stage34HandoffError(
                    f"selected variant {v.variant_id} has non-viable status {analysis.overall_status}"
                )
            risk = risk_by_id.get(v.variant_id)
            if risk is None:
                raise Stage34HandoffError(
                    f"selected variant {v.variant_id} has no VariantRiskReport"
                )
            variant_inputs.append(Stage35VariantInput(
                variant=v,
                transfer_analysis=analysis,
                risk_report=risk,
                experiment_resolvable=[
                    u for u in handoff.experiment_resolvable_dimensions
                    if u.variant_id == v.variant_id
                ],
            ))

        return Stage35Input(
            run_id=handoff.run_id,
            confirmed_idea=handoff.confirmed_idea,
            transfer_analysis=handoff.transfer_analysis,
            transfer_constraints=handoff.transfer_constraints,
            variants=variant_inputs,
            nonblocking_warnings=handoff.nonblocking_warnings,
        )

    confirmed_idea: IdeaContract

    transfer_constraints: list[TransferConstraint]

    nonblocking_warnings: list[UnresolvedDimension]

class SharedExperimentProtocol(BaseModel):
    """所有 Variant 共享的实验协议。"""

    model_config = ConfigDict(extra="forbid")

    protocol_id: str
    schema_version: Literal[1]
    protocol_version: int = 1

    # --- 输入身份（planning subset，不含 command_sha256）---
    planning_input_refs: PlanningInputRefs

    # --- 评价链补充引用 ---
    supplemental_refs: SupplementalEvaluationRefs

    # --- 当前评价协议标识 ---
    evaluation_protocol_ref: ArtifactRef
    # 指向评价器代码/配置的 artifact 引用
    # 用于 baseline 复用时与 BaselineResultRef.evaluation_contract_sha256 比较

    # --- Baseline ---
    baseline_method: str
    baseline_config_sha256: str
    baseline_policy: BaselineExecutionPolicy

    # --- 共享实验参数（唯一主源）---
    seeds: list[int] = Field(min_length=1)
    # MatrixBuilder 使用 seeds[0] —— 非空由 schema 保证
    primary_metric: str
    metric_direction: Literal["maximize", "minimize"]

    # --- 共享约束（来自 3.4 transfer_constraints）---
    protected_paths: list[str]
    must_not_change: list[InterfaceConstraint]

    # --- 证据 ---
    protocol_evidence_ids: list[str]
    protocol_fingerprint: str

    @model_validator(mode="after")
    def _validate_seeds(self):
        if len(self.seeds) != len(set(self.seeds)):
            raise ValueError("seeds must be unique")
        if self.baseline_policy.seeds != self.seeds:
            raise ValueError("baseline_policy.seeds must equal protocol.seeds")
        source = self.baseline_policy.reuse_source
        if source is not None:
            if source.seeds != self.seeds:
                raise ValueError("reused baseline seeds must equal protocol seeds")
            if sorted(source.completed_seed_ids) != sorted(self.seeds):
                raise ValueError("reused baseline must complete every protocol seed")
            # per_seed_metrics 必须覆盖每个 seed，且 primary_metric 完整、唯一、有限
            primary_metrics = [
                m for m in source.per_seed_metrics
                if m.metric_name == self.primary_metric
            ]
            metric_seeds = {m.seed for m in primary_metrics}
            if metric_seeds != set(self.seeds):
                raise ValueError(
                    "reused baseline per_seed_metrics must cover every protocol seed "
                    f"for primary_metric '{self.primary_metric}'"
                )
            if len(primary_metrics) != len(metric_seeds):
                raise ValueError("duplicate seed in per_seed_metrics for primary_metric")
            for m in primary_metrics:
                if m.metric_value is None or math.isnan(m.metric_value) or math.isinf(m.metric_value):
                    raise ValueError(f"seed {m.seed} primary_metric is non-finite")
        return self

## 3.3 规则覆盖要求

```python
def validate_decision_rule_coverage(rules: list[ScientificDecisionRule]) -> list[ValidationIssue]:
    """验证规则有序、完备覆盖。

    评估语义：first-match priority。按 priority 从小到大依次匹配，
    第一个条件满足的规则即为最终结论。规则可重叠，顺序决定唯一结果。
    """
    issues = []

    # 1. 优先级不重复
    priorities = [r.priority for r in rules]
    if len(priorities) != len(set(priorities)):
        issues.append(ValidationIssue(severity="blocking", message="Duplicate rule priorities"))

    # 2. 所有 ScientificConclusion 值至少被一条规则覆盖
    covered_conclusions = {r.conclusion_code for r in rules}
    missing = set(ScientificConclusion.__members__.values()) - covered_conclusions
    for m in missing:
        issues.append(ValidationIssue(
            severity="blocking",
            message=f"ScientificConclusion.{m.name} not covered by any rule"
        ))

    # 3. 恰好一个 ALWAYS 兜底规则，优先级数字最大
    always_rules = [r for r in rules if r.condition.condition_type == DecisionConditionType.ALWAYS]
    fallback_rule = None
    if len(always_rules) != 1:
        issues.append(ValidationIssue(
            severity="blocking",
            message=f"Exactly one ALWAYS catch-all required, found {len(always_rules)}"
        ))
    else:
        fallback_rule = always_rules[0]
        if fallback_rule.priority != max(r.priority for r in rules):
            issues.append(ValidationIssue(severity="blocking",
                message="ALWAYS catch-all must have the lowest priority (largest number)"))
        if fallback_rule.conclusion_code != ScientificConclusion.MIXED:
            issues.append(ValidationIssue(severity="blocking",
                message="ALWAYS catch-all must conclude MIXED"))

    # 4. INCOMPLETE 的 priority 必须严格小于所有非兜底结论规则的 priority
    incomplete_rules = [r for r in rules if r.conclusion_code == ScientificConclusion.INCOMPLETE]
    other_rules = [r for r in rules
                   if (fallback_rule is None or r != fallback_rule)
                   and r not in incomplete_rules]
    if incomplete_rules and other_rules:
        max_incomplete_priority = max(r.priority for r in incomplete_rules)
        min_other_priority = min(r.priority for r in other_rules)
        if max_incomplete_priority >= min_other_priority:
            issues.append(ValidationIssue(
                severity="blocking",
                message="All INCOMPLETE rules must have higher priority (smaller number) "
                        "than any BENEFICIAL/WORSE/EQUIVALENT rule"
            ))

    return issues
```

附带 StatisticalAnalysisPlan 本身的证据校验：

```python
def validate_stat_plan(
    plan: StatisticalAnalysisPlan,
    evidence_index: EvidenceIndex,
) -> list[ValidationIssue]:
    issues = []
    if plan.minimum_meaningful_effect is not None:
        has_valid_evidence = (
            bool(plan.minimum_meaningful_effect_evidence_ids)
            and all(
                evidence_index.exists(eid)
                for eid in plan.minimum_meaningful_effect_evidence_ids
            )
        )
        has_valid_user_confirmation = (
            plan.user_confirmation_evidence_id is not None
            and evidence_index.exists(plan.user_confirmation_evidence_id)
        )
        if not has_valid_evidence and not has_valid_user_confirmation:
            issues.append(ValidationIssue(
                severity="blocking",
                message="minimum_meaningful_effect set but no valid evidence_ids "
                        "(must exist in EvidenceIndex) or user_confirmation_evidence_id"
            ))
    return issues
```

推荐优先级顺序（数字越小越先匹配，first-match priority）：

```
10  INCOMPLETE                ← 种子不足时最先判定
20  BENEFICIAL                ← 全部种子改善
30  WORSE                     ← 全部种子退化
40  PRACTICALLY_EQUIVALENT    ← 差异在等价区间内
50  MIXED (condition=ALWAYS)  ← 兜底 catch-all

# 4. Step 3：ExperimentTrialSpecs — 试验规格

## 4.1 设计原则

```
1. 所有 intent（baseline / fit / smoke / full）统一存放在 ExperimentTrialSpecs
2. TrialIntent 不含 depends_on、不含 timeout_policy_ref、不含 shell 命令

# 5. Step 4：ExperimentMatrix — 实验矩阵

## 5.1 设计原则

```
1. 不含资源估算数字——资源只在 resource_budget.json
2. 依赖关系只在这里表达——TrialIntent 不含 depends_on
3. baseline × seeds + variant × fit/smoke/full + variant × seeds
4. 所有 entry 的 variant_id 可以是 None（baseline）或 str（variant）
```

## 5.2 Schema

```python
class ExperimentMatrix(BaseModel):
    """baseline × seeds + variant × (fit + smoke) + variant × seeds 矩阵。"""

    model_config = ConfigDict(extra="forbid")

    matrix_id: str
    schema_version: Literal[1]
    protocol_fingerprint: str

    seeds: list[int]                           # 引用 SharedExperimentProtocol.seeds
    variants: list[str]                        # variant_ids

    entries: list[MatrixEntry]
    input_bindings: list[MatrixInputBinding] = Field(default_factory=list)

class MatrixEntry(BaseModel):
    entry_id: str                              # e.g. "baseline_s42", "var_A_fit", "var_A_smoke", "var_A_full_s42"
    variant_id: str | None                     # None = baseline entry
    stage: Literal["baseline", "fit", "smoke", "full"]
    seed: int | None                           # fit/smoke 固定 seed，仍记录具体值

    intent_ref: str                            # 引用 TrialIntent.intent_id

    depends_on: list[str]                      # 依赖的 entry_ids
    # 依赖关系唯一主源——TrialIntent 中不重复

    expected_outputs: list[ArtifactProduction] = Field(default_factory=list)
    # 每个 production 有唯一 production_id，供 MatrixInputBinding 精确引用

    shared_axes: list[str]
    independent_axes: list[str]

    priority: int = 0                          # 0 = 最高

class MatrixInputBinding(BaseModel):
    """表达 artifact 产物来源（不仅拓扑顺序，精确到生产 ID）。

    例如：per_evaluation_seed 下 full_s42 的模型权重来自 fit_s42 的 best_model 产出。
    """

    consumer_entry_id: str
    consumer_requirement_id: str               # 对应 TrialIntent.required_inputs 中的项目
    producer_entry_id: str
    producer_production_id: str                # 对应 producer expected_outputs 中的 production_id


class ArtifactProduction(BaseModel):
    """一个实验 entry 产生的确定 artifact。"""
    production_id: str                         # e.g. "best_model", "last_checkpoint", "ema_weights"
    artifact_type: Literal["model_weights", "checkpoint", "metrics_json",
                           "predictions", "embeddings", "logs"]
    description: str
```

## 5.3 展开规则

```python
class MatrixBuilder:
    def build(self, protocol: SharedExperimentProtocol,
              specs: ExperimentTrialSpecs) -> ExperimentMatrix:
        entries = []
        seeds = protocol.seeds
        baseline_policy = protocol.baseline_policy

        # 1. Baseline entries（仅 run_fresh 时）
        if baseline_policy.mode == "run_fresh":
            assert specs.baseline is not None, "baseline intent required for run_fresh"
            for seed in seeds:
                entries.append(MatrixEntry(
                    entry_id=f"baseline_s{seed}",
                    variant_id=None,
                    stage="baseline",
                    seed=seed,
                    intent_ref=specs.baseline.intent_id,
                    depends_on=[],
                    shared_axes=["protocol", "dataset", "metric"],
                    independent_axes=[f"seed_{seed}"],
                    priority=0,
                ))

        # 2. Variant entries
        for variant in specs.variants:
            # 2a. Fit（可选）
            fit_entries = []
            fit_policy = variant.fit_seed_policy
            if variant.fit is not None:
                assert fit_policy is not None, "fit_seed_policy required when fit exists"
                if fit_policy == "shared_fixed":
                    # 一次 fit，固定 seed
                    fit_seed = seeds[0]
                    fit_entry_id = f"{variant.variant_id}_fit_s{fit_seed}"
                    fit_entries.append(MatrixEntry(
                        entry_id=fit_entry_id,
                        variant_id=variant.variant_id,
                        stage="fit",
                        seed=fit_seed,
                        intent_ref=variant.fit.intent_id,
                        depends_on=[],  # fit 不依赖 baseline evaluation
                        shared_axes=["protocol", "dataset", "baseline"],
                        independent_axes=["variant_modification", "fit_stage", "fit_seed_shared"],
                        expected_outputs=[ArtifactProduction(
                            production_id="best_model",
                            artifact_type="model_weights",
                            description="fit 产出的模型权重",
                        )],
                        priority=0,
                    ))
                elif fit_policy == "per_evaluation_seed":
                    # 每个 evaluation seed 独立 fit
                    # fit_s42 → full_s42, fit_s43 → full_s43, ...
                    for seed in seeds:
                        fit_entry_id = f"{variant.variant_id}_fit_s{seed}"
                        fit_entries.append(MatrixEntry(
                            entry_id=fit_entry_id,
                            variant_id=variant.variant_id,
                            stage="fit",
                            seed=seed,
                            intent_ref=variant.fit.intent_id,
                            depends_on=[],  # fit 不依赖 baseline evaluation
                            shared_axes=["protocol", "dataset", "baseline"],
                            independent_axes=["variant_modification", "fit_stage", f"seed_{seed}"],
                            expected_outputs=[ArtifactProduction(
                                production_id="best_model",
                                artifact_type="model_weights",
                                description=f"fit_s{seed} 产出的模型权重",
                            )],
                            priority=0,
                        ))
                elif fit_policy == "deterministic_no_seed":
                    fit_entry_id = f"{variant.variant_id}_fit_det"
                    fit_entries.append(MatrixEntry(
                        entry_id=fit_entry_id,
                        variant_id=variant.variant_id,
                        stage="fit",
                        seed=None,             # 确定性训练，不消耗 seed
                        intent_ref=variant.fit.intent_id,
                        depends_on=[],
                        shared_axes=["protocol", "dataset", "baseline"],
                        independent_axes=["variant_modification", "fit_stage", "fit_deterministic"],
                        expected_outputs=[ArtifactProduction(
                            production_id="best_model",
                            artifact_type="model_weights",
                            description="确定性 fit 产出的模型权重",
                        )],
                        priority=0,
                    ))
                entries.extend(fit_entries)

            # 2b. Smoke（按 fit_policy 决定依赖和数量）
            if fit_policy in ("shared_fixed", "deterministic_no_seed"):
                # 一个 smoke 依赖所有 fit entries
                smoke_seed = seeds[0]
                smoke_entry_id = f"{variant.variant_id}_smoke"
                smoke_deps = [e.entry_id for e in fit_entries]
                entries.append(MatrixEntry(
                    entry_id=smoke_entry_id,
                    variant_id=variant.variant_id,
                    stage="smoke",
                    seed=smoke_seed,
                    intent_ref=variant.smoke.intent_id,
                    depends_on=smoke_deps,
                    shared_axes=["protocol", "dataset", "metric", "baseline"],
                    independent_axes=["variant_modification", "smoke_check"],
                    priority=1,
                ))
                smoke_entry_ids = [smoke_entry_id]
            elif fit_policy == "per_evaluation_seed":
                # 每 seed 独立 smoke，只依赖对应 fit
                smoke_entry_ids = []
                for seed in seeds:
                    sid = f"{variant.variant_id}_smoke_s{seed}"
                    smoke_entry_ids.append(sid)
                    entries.append(MatrixEntry(
                        entry_id=sid,
                        variant_id=variant.variant_id,
                        stage="smoke",
                        seed=seed,
                        intent_ref=variant.smoke.intent_id,
                        depends_on=[f"{variant.variant_id}_fit_s{seed}"],
                        shared_axes=["protocol", "dataset", "metric", "baseline"],
                        independent_axes=["variant_modification", "smoke_check", f"seed_{seed}"],
                        priority=1,
                    ))
            else:
                # 无 fit：smoke 只做代码可运行性检查，不依赖 baseline evaluation
                smoke_seed = seeds[0]
                smoke_entry_id = f"{variant.variant_id}_smoke"
                smoke_deps = []
                entries.append(MatrixEntry(
                    entry_id=smoke_entry_id,
                    variant_id=variant.variant_id,
                    stage="smoke",
                    seed=smoke_seed,
                    intent_ref=variant.smoke.intent_id,
                    depends_on=smoke_deps,
                    shared_axes=["protocol", "dataset", "metric", "baseline"],
                    independent_axes=["variant_modification", "smoke_check"],
                    priority=1,
                ))
                smoke_entry_ids = [smoke_entry_id]

            # 2c. Full（每个 seed，按 fit_seed_policy 绑定对应产物）
            for seed in seeds:
                full_deps = [smoke_entry_ids[0]] if fit_policy != "per_evaluation_seed" else [f"{variant.variant_id}_smoke_s{seed}"]
                if fit_policy == "per_evaluation_seed":
                    fit_entry_id = f"{variant.variant_id}_fit_s{seed}"
                    full_deps.append(fit_entry_id)
                elif fit_policy == "shared_fixed":
                    # full_s42/s43/s44 共享同一 fit 产物
                    full_deps.append(f"{variant.variant_id}_fit_s{seeds[0]}")

                entries.append(MatrixEntry(
                    entry_id=f"{variant.variant_id}_full_s{seed}",
                    variant_id=variant.variant_id,
                    stage="full",
                    seed=seed,
                    intent_ref=variant.full.intent_id,
                    depends_on=full_deps,
                    shared_axes=["protocol", "dataset", "metric", "baseline"],
                    independent_axes=["variant_modification", f"seed_{seed}"],
                    priority=2,
                ))

        # 3. 生成 artifact bindings（fit → smoke → full 链）
        bindings = []
        for variant in specs.variants:
            if variant.fit is not None:
                def _require_input(intent, artifact_type, variant_id, intent_name):
                    req = next(
                        (r for r in intent.required_inputs
                         if r.artifact_type == artifact_type),
                        None
                    )
                    if req is None:
                        raise MatrixBuildError(
                            f"variant {variant_id}: {intent_name} intent must declare "
                            f"a {artifact_type} requirement when fit exists"
                        )
                    return req

                smoke_weight_req = _require_input(
                    variant.smoke, "model_weights", variant.variant_id, "smoke"
                )
                full_weight_req = _require_input(
                    variant.full, "model_weights", variant.variant_id, "full"
                )
                # Smoke bindings（按 fit_seed_policy）
                if variant.fit_seed_policy == "per_evaluation_seed":
                    for seed in seeds:
                        bindings.append(MatrixInputBinding(
                            consumer_entry_id=f"{variant.variant_id}_smoke_s{seed}",
                            consumer_requirement_id=smoke_weight_req.requirement_id,
                            producer_entry_id=f"{variant.variant_id}_fit_s{seed}",
                            producer_production_id="best_model",
                        ))
                else:
                    smoke_id = f"{variant.variant_id}_smoke"
                    fit_id = (f"{variant.variant_id}_fit_s{seeds[0]}"
                              if variant.fit_seed_policy == "shared_fixed"
                              else f"{variant.variant_id}_fit_det")
                    bindings.append(MatrixInputBinding(
                        consumer_entry_id=smoke_id,
                        consumer_requirement_id=smoke_weight_req.requirement_id,
                        producer_entry_id=fit_id,
                        producer_production_id="best_model",
                    ))
                # Full bindings
                for full_seed in seeds:
                    full_entry = f"{variant.variant_id}_full_s{full_seed}"
                    if variant.fit_seed_policy == "per_evaluation_seed":
                        fit_entry = f"{variant.variant_id}_fit_s{full_seed}"
                    elif variant.fit_seed_policy == "shared_fixed":
                        fit_entry = f"{variant.variant_id}_fit_s{seeds[0]}"
                    else:
                        fit_entry = f"{variant.variant_id}_fit_det"
                    bindings.append(MatrixInputBinding(
                        consumer_entry_id=full_entry,
                        consumer_requirement_id=full_weight_req.requirement_id,
                        producer_entry_id=fit_entry,
                        producer_production_id="best_model",
                    ))

        return ExperimentMatrix(
            matrix_id=f"matrix_{protocol.protocol_id}",
            schema_version=1,
            protocol_fingerprint=protocol.protocol_fingerprint,
            seeds=seeds,
            variants=[v.variant_id for v in specs.variants],
            entries=entries,
            input_bindings=bindings,
        )
```

```python
class ExperimentalResolutionPlans(BaseModel):
    """将 3.4 的未解决问题编译为可执行的验证步骤。"""

class ExperimentalResolutionPlan(BaseModel):
    unresolved_dimension_id: str
    dimension: str
    variant_id: str
    verification_stage: Literal["fit", "smoke", "full"]

    target_entry_ids: list[str]               # Matrix 生成后绑定

    observable: str
    observation_source: str

    acceptance_criterion: ResolutionCriterion
    rejection_criterion: ResolutionCriterion | None = None
    # discriminator 由 ResolutionCriterion 的 Annotated Union 定义，不在此重复

    result_on_accept: ResolutionOutcome
    result_on_reject: ResolutionOutcome | None = None
    result_on_inconclusive: ResolutionOutcome = ResolutionOutcome.INCONCLUSIVE

    @model_validator(mode="after")
    def _validate_outcomes(self):
        # rejection_criterion 和 result_on_reject 必须成对出现
        if (self.rejection_criterion is None) != (self.result_on_reject is None):
            raise ValueError(
                "rejection_criterion and result_on_reject must appear together"
            )
        if self.result_on_accept == ResolutionOutcome.INCONCLUSIVE:
            raise ValueError("accept branch cannot resolve to inconclusive")
        return self

class ThresholdCriterion(BaseModel):

# 7. Step 6：ResourceBudget + BudgetDecision — 资源预算与决策

## 7.1 设计原则

```
1. 资源估算包含区间（low/high）+ planning_value + safety_factor
2. 低置信估算使用上界进行预算判断
3. 所有资源数字只在 resource_budget.json——Matrix 不含资源字段
4. Budget revision 生命周期闭环：protocol version+1 → 重生成 2–6
```

        if self.status == "within_budget" and self.over_budget_items:
            raise ValueError("within_budget cannot contain over_budget_items")
        return self

## 7.3 Budget Revision 版本化闭环

```text
revision_required
    ↓
用户选择 revision_option
    ↓
revision_selected → selected_revision_option_id 记录
    ↓
应用 revision：
  1. SharedExperimentProtocol.protocol_version += 1
  2. 如果 affects_seeds → 更新 protocol.seeds → 重新生成 StatisticalAnalysisPlan（检查兼容性）
  3. 如果 affects_variants → 更新 variant_trial_specs.json
  4. 重新生成 ExperimentMatrix
  5. 重新生成 ExperimentalResolutionPlans
  6. 重新计算 ResourceBudget
  7. 旧 downstream artifacts 全部失效
  8. 新 budget_decision.status 重新计算
    ↓
within_budget → Handoff
或再次 revision_required → 循环（最多 3 次）
```

**进入 3.6 的必要条件：`status == within_budget` 或 `status == override_confirmed`**

**禁止 handoff：`revision_required`、`revision_selected`（未应用）、`override_pending`、`override_rejected`**

def validate_evaluation_chain_coverage(
    protocol: SharedExperimentProtocol,
    evidence_index: EvidenceIndex,
    repository_artifact_index: RepositoryArtifactIndex,
    dataset_manifest_index: DatasetManifestIndex,
) -> list[ValidationIssue]:
    """基于已有 artifact 和 EvidenceRef 判断，不读取仓库文件。"""
    issues = []
    refs = protocol.supplemental_refs

    components = [
        ("evaluator", refs.evaluator_ref, refs.evaluator_coverage_evidence_ids,
         _is_evaluator_in_repository(protocol, repository_artifact_index)),
        ("metric_parser", refs.metric_parser_ref, refs.metric_parser_coverage_evidence_ids,
         _is_metric_parser_in_repository(protocol, repository_artifact_index)),
        ("postprocessing", refs.postprocessing_ref, refs.postprocessing_coverage_evidence_ids,
         _is_postprocessing_covered_by_config(protocol, repository_artifact_index)),
        ("dataset_split", refs.dataset_split_ref, refs.dataset_split_coverage_evidence_ids,
         _is_split_in_manifest(protocol, dataset_manifest_index)),
    ]

    for name, explicit_ref, evidence_ids, covered_by_fingerprint in components:
        has_explicit_ref = explicit_ref is not None
        has_evidence = all(evidence_index.exists(eid) for eid in evidence_ids) if evidence_ids else False

            issues.append(ValidationIssue(
                severity="blocking",
                message=f"{name} not covered by planning_input_refs and no supplemental ref"
            ))
        if has_explicit_ref and not has_evidence:
            issues.append(ValidationIssue(
                severity="blocking",
                message=f"{name} has supplemental ref but coverage evidence missing"
            ))

    return issues

```python
class ExperimentPlannerHandoff(BaseModel):
    """3.5 → 3.6 handoff。SHA-only 引用。"""

    model_config = ConfigDict(extra="forbid")

    schema_version: Literal[1]
    run_id: str
    source_handoff_sha256: str
    artifact_manifest: ArtifactManifest
    # 引用 artifact 1–8 的 SHA。Handoff 自身（artifact 9）不列入 manifest
    selected_variant_ids: list[str]
    validation_report_sha256: str
    next_stage: Literal["3.6_patch_planner"]

class HandoffEmitter:
    """生成 Handoff 前强制执行所有门禁。"""

    def emit(self, artifacts: dict[str, Path], run_id: str,
             source_sha256: str) -> ExperimentPlannerHandoff:
        # Gate 1: validation report 存在且 status=passed
        report = ExperimentPlanValidationReport.model_validate_json(
            artifacts["experiment_plan_validation_report.json"].read_text()
        )
        if report.status != "passed":
            raise HandoffBlockedError("validation report not passed")

        # Gate 1b (TOCTOU): report 引用的 artifact SHA 必须匹配当前 artifact 1–7
        current_sha = {name: _sha_file(artifacts[name]) for name in _PLANNING_FILES}
        reported_refs = {r.relative_path: r.sha256 for r in report.validated_artifact_refs}
        for name, sha in current_sha.items():
            if reported_refs.get(name) != sha:
                raise HandoffBlockedError(
                    f"validation report binds to different {name} SHA "
                    f"(report={reported_refs.get(name)}, current={sha})"
                )

        # Gate 1c: protocol_fingerprint 跨 artifact 一致
        protocol = SharedExperimentProtocol.model_validate_json(
            artifacts["shared_experiment_protocol.json"].read_text()
        )
        specs = ExperimentTrialSpecs.model_validate_json(
            artifacts["experiment_trial_specs.json"].read_text()
        )
        matrix = ExperimentMatrix.model_validate_json(
            artifacts["experiment_matrix.json"].read_text()
        )
        budget = ResourceBudget.model_validate_json(
            artifacts["resource_budget.json"].read_text()
        )
        fp = protocol.protocol_fingerprint
        for name, obj in [("specs", specs), ("matrix", matrix), ("budget", budget)]:
            if getattr(obj, "protocol_fingerprint", None) != fp:
                raise HandoffBlockedError(f"{name}.protocol_fingerprint mismatch")

        # Gate 2: budget 必须为 within_budget 或 override_confirmed
        if budget.budget_decision.status not in ("within_budget", "override_confirmed"):
            raise HandoffBlockedError(
                f"budget status {budget.budget_decision.status} not handoff-ready"
            )

        # Gate 3: manifest 恰好包含 artifact 1–8
        manifest = ArtifactManifest(entries=[
            ManifestEntry(relative_path=name, sha256=sha, artifact_type=name)
            for name, sha in current_sha.items()
        ] + [
            ManifestEntry(
                relative_path="experiment_plan_validation_report.json",
                sha256=_sha_file(artifacts["experiment_plan_validation_report.json"]),
                artifact_type="validation_report",
            )
        ])

        # Gate 4: selected_variant_ids 从 specs 派生，并与 matrix 一致
        selected_variant_ids = [v.variant_id for v in specs.variants]
        if matrix.variants != selected_variant_ids:
            raise HandoffBlockedError("matrix.variants mismatch with trial specs")

        return ExperimentPlannerHandoff(
            schema_version=1,
            run_id=run_id,
            source_handoff_sha256=source_sha256,
            artifact_manifest=manifest,
            selected_variant_ids=selected_variant_ids,
            validation_report_sha256=_sha_file(
                artifacts["experiment_plan_validation_report.json"]
            ),
            next_stage="3.6_patch_planner",
        )

  [ ] minimum_meaningful_effect 无 evidence 且无用户确认 → blocking，或 forced descriptive-only

  [ ] DatasetPartitionRef.declared_role == "test" → 拒绝
  [ ] 不同 Variant search_budget 不一致 → blocking
  [ ] 无 validation split + mode=predeclared_search → 拒绝

  [ ] MatrixEntry 不含 estimated_gpu_hours
  [ ] depends_on 无环
  [ ] fit entry 的 depends_on 正确（baseline 或空）
  [ ] smoke entry 的 depends_on 包含 fit（如有），不包含 baseline（无数据依赖）
  [ ] full entry 的 depends_on 包含 smoke

Resolution Coverage:
  [ ] 每个 3.4 unresolved dimension → 恰好一个 ExperimentalResolutionPlan
  [ ] 所有 target_entry_ids 存在于 experiment_matrix

Budget:
  [ ] planning_value >= estimated_gpu_hours_high * safety_factor（confidence=low 时必须满足）
  [ ] revision_selected → selected_revision_option_id 非空
  [ ] revision 应用后 protocol_version 递增

  [ ] 所有 target_entry_ids 存在于 experiment_matrix 或为 "*"

  [ ] relative_path 拒绝 ".." 和绝对路径
