```python
class CurrentRunBaselineMetricRef(BaseModel):
    source_type: Literal["current_run"]
    unit_id: str
```

## 1.2 PairedMetricObservation（seed-keyed，支持 reused baseline）

```python
class PairedMetricObservation(BaseModel):
    seed: int

    baseline_source: BaselineMetricSource
    baseline_value: float  # ← Validator 从 baseline_source.metric_ref 重算

    variant_unit_id: str
    variant_metric_ref: ArtifactRef
    variant_value: float  # ← Validator 从 variant_metric_ref 重算

    delta: float  # ← Validator 重算
    baseline_validity_ref: ArtifactRef
    variant_validity_ref: ArtifactRef
    protocol_fingerprint: str

class AggregatedMetricComparison(BaseModel):
    variant_id: str
    metric_name: str
    dataset_row: str

    paired_observations: list[PairedMetricObservation]
    aggregation_method: str

    baseline_aggregate: float | None
    variant_aggregate: float | None
    dispersion: float | None

    effect_size: float | None
    relative_change_pct: float | None

    comparison_status: Literal[
        "comparable",
        "missing_baseline", "missing_variant",
        "invalid_baseline", "invalid_variant",
        "insufficient_pairs",
    ]
```

Validator 必须从 `ParsedMetric` artifact 重算 `baseline_value`、`variant_value`、`delta`，不一致则拒绝。

## 1.5 Idea-support 决策表（first-match priority，修正输入集合）

```
输入: 每个 Variant 的 ScientificConclusion (3.5 enum)
输出: idea_support (str)

1. 存在 beneficial 且 存在 worse
   → implementation_sensitive

2. 全部 beneficial
   → consistently_supported

3. 至少一个 beneficial，且无 worse
   → supported_by_at_least_one_variant

4. 全部 worse
   → not_supported_by_tested_variants

5. 全部 incomplete
   → cannot_judge

6. 存在 worse，无 beneficial，存在 practically_equivalent / mixed
   → not_supported_or_not_demonstrated

7. 存在 practically_equivalent / mixed，无 beneficial，无 worse
   → not_demonstrated_or_partial

8. 其他
   → cannot_judge
```

规则 6 保证 "worse+equivalent，无 beneficial" 不掉进兜底 cannot_judge。

## 1.7 ResourceComparison（逐 Variant 状态）

```python
class ResourceComparison(BaseModel):
    variant_id: str
    dimension: Literal["wall_time", "peak_rss", "peak_gpu_memory", "gpu_seconds"]
    comparison_status: Literal[
        "comparable",
        "missing_baseline", "missing_variant",
        "measurement_unavailable",
        "incompatible_measurement",
    ]
    baseline_value: float | None
    variant_value: float | None
    delta: float | None

class VariantResourceComparison(BaseModel):
    variant_id: str
    comparisons: list[ResourceComparison]
    budget_status: Literal["within_budget", "over_budget", "not_assessable"]

class ResourceComparisonReport(BaseModel):
    per_variant: dict[str, VariantResourceComparison]
    measurement_quality: Literal["complete", "partial", "unavailable"]
```

`budget_status` 下沉到逐 Variant，不设单一 run 级。

# 2. 测试（新增）

```
[ ] reused baseline → ReusedBaselineMetricRef 含 source_run_id + identity_validation_ref
[ ] current_run baseline → CurrentRunBaselineMetricRef
[ ] PairedMetricObservation 的 value/delta 由 Validator 从 artifact 重算
[ ] Validator 检测缓存值与 artifact 不一致 → 拒绝
[ ] Variant A within_budget + Variant B over_budget → 各自独立，不合并
[ ] worse+equivalent 无 beneficial → not_supported_or_not_demonstrated（rule 6）
[ ] beneficial+equivalent 无 worse → supported_by_at_least_one_variant（rule 3）
[ ] baseline_source=reused → 报告中标记 baseline_source=reused
```
