# 1. 核心模型

## 1.1 Delta 语义

```python
def compute_deltas(baseline: float, variant: float, direction: Literal["maximize", "minimize"]):
    raw_delta = variant - baseline
    if direction == "maximize":
        improvement_delta = raw_delta
    else:
        improvement_delta = -raw_delta
    relative_change_pct = (raw_delta / baseline * 100) if baseline != 0 else None
    return raw_delta, improvement_delta, relative_change_pct
```

`improvement_delta > 0` 总是表示 improvement。3.5 的所有 `all_seeds_improved`、`all_seeds_degraded`、`mean_improved_above_threshold` 全部基于 `improvement_delta`。

## 1.3 PairedMetricObservation

```python
class PairedMetricObservation(BaseModel):
    seed: int
    baseline_source: BaselineMetricSource
    baseline_value: float  # Validator 从 baseline_source.metric_ref 重算

    variant_unit_id: str
    variant_metric_ref: ArtifactRef
    variant_value: float  # Validator 从 variant_metric_ref 重算

    direction: Literal["maximize", "minimize"]
    raw_delta: float         # variant - baseline
    improvement_delta: float  # direction-aware
    relative_change_pct: float | None  # None if baseline==0

    variant_validity_ref: ArtifactRef
    protocol_fingerprint: str
```

Validator 检查：
```
observation.seed == baseline_source.seed
observation.seed == variant unit.seed
baseline_value == dereference(baseline_source.metric_ref).value
variant_value == dereference(variant_metric_ref).value
raw_delta == variant_value - baseline_value
improvement_delta == direction-aware(raw_delta, direction)
relative_change_pct 正确计算，baseline=0 时为 None
```

## 1.6 IdeaSupportConclusion（enum + first-match table）

```python
class IdeaSupportConclusion(str, Enum):
    IMPLEMENTATION_SENSITIVE = "implementation_sensitive"
    CONSISTENTLY_SUPPORTED = "consistently_supported"
    SUPPORTED_BY_AT_LEAST_ONE = "supported_by_at_least_one_variant"
    NOT_SUPPORTED_BY_TESTED = "not_supported_by_tested_variants"
    CANNOT_JUDGE = "cannot_judge"
    NOT_SUPPORTED_OR_NOT_DEMONSTRATED = "not_supported_or_not_demonstrated"
    NOT_DEMONSTRATED_OR_PARTIAL = "not_demonstrated_or_partial"

def derive_idea_support(conclusions: list[VariantScientificConclusion]) -> IdeaSupportConclusion:
    statuses = [c.conclusion for c in conclusions]
    has_beneficial = ScientificConclusion.BENEFICIAL in statuses
    has_worse = ScientificConclusion.WORSE in statuses
    has_equivalent = ScientificConclusion.PRACTICALLY_EQUIVALENT in statuses
    has_mixed = ScientificConclusion.MIXED in statuses
    all_bene = all(s == ScientificConclusion.BENEFICIAL for s in statuses)
    all_worse = all(s == ScientificConclusion.WORSE for s in statuses)
    all_incomplete = all(s == ScientificConclusion.INCOMPLETE for s in statuses)

    if has_beneficial and has_worse:
        return IdeaSupportConclusion.IMPLEMENTATION_SENSITIVE        # 1
    if all_bene:
        return IdeaSupportConclusion.CONSISTENTLY_SUPPORTED          # 2
    if has_beneficial and not has_worse:
        return IdeaSupportConclusion.SUPPORTED_BY_AT_LEAST_ONE       # 3
    if all_worse:
        return IdeaSupportConclusion.NOT_SUPPORTED_BY_TESTED         # 4
    if all_incomplete:
        return IdeaSupportConclusion.CANNOT_JUDGE                    # 5
    if has_worse and not has_beneficial and (has_equivalent or has_mixed):
        return IdeaSupportConclusion.NOT_SUPPORTED_OR_NOT_DEMONSTRATED  # 6
    if not has_beneficial and not has_worse and (has_equivalent or has_mixed):
        return IdeaSupportConclusion.NOT_DEMONSTRATED_OR_PARTIAL     # 7
    return IdeaSupportConclusion.CANNOT_JUDGE                        # 8
```

# 4. 测试

```
[ ] baseline=0 → relative_change_pct=None（不产生 ±inf）
[ ] maximize: variant>baseline → improvement_delta>0
[ ] minimize: variant<baseline → improvement_delta>0
[ ] PairedMetricObservation values 由 Validator 从 artifact 重算，不一致拒绝
[ ] observation.seed == baseline_source.seed == variant unit.seed
[ ] IdeaSupportConclusion 为 enum，不输出任意字符串
[ ] beneficial+worse → IMPLEMENTATION_SENSITIVE (rule 1)
[ ] all_beneficial → CONSISTENTLY_SUPPORTED (rule 2)
[ ] worse+equivalent → NOT_SUPPORTED_OR_NOT_DEMONSTRATED (rule 6)
[ ] per-variant budget_status 独立，不合并
[ ] 无 resource_budget_ref → budget_status=not_assessable
[ ] reused baseline → ReusedBaselineMetricRef + identity_validation_ref
[ ] reused baseline 报告中标记 baseline_source=reused
[ ] final_report 篡改结论 → ReportFactValidator 拒绝
[ ] output: scientific_validity_aggregate.json (not .*_report.json)
```
