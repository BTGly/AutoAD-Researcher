# 0. 阶段定义

```
ExperimentPlannerHandoff (3.5) + PatchRunnerHandoff v2 (3.7, schema_version=2)
    ↓
Runner Intake Validator → RunnerIntakeReport
    ↓
Matrix → Immutable ExecutionUnitPlans
    ↓
每个 ExecutionUnit:
  → ExperimentCommandPlan (3.0) → Preflight (3.0)
  → Attempt 01: AttemptIdentitySnapshot → ResourceUsageReport →
    MetricsReport → ScientificValidityReport → RetryDecision
  → Attempt 02 (仅 retry_same_command 且 identity 匹配)
    ↓
ExecutionUnitRecord + ExecutionManifest
    ↓
ExperimentExecutionHandoff → 3.9
```

## 0.1 关键原则

- `ExecutionUnitPlan` 不可变，SHA 一经计算永不改变
- 运行时解析结果入 `AttemptRecord`，不入 Plan
- 计划态 binding 有唯一 `binding_id`
- Retry = 失败分类允许 + identity 匹配
- 所有 schema Validator 使用 `raise ValueError`，禁止 `assert`
- PatchRunnerHandoff v2 集合 Validator 要求 selected ∩ workspace_variants 精确相等

```python
class WorkspaceExecutionRef(BaseModel):
    workspace_id: str
    subject_type: Literal["baseline", "variant"]
    variant_ids: list[str]
    repository_fingerprint: str
    repository_commit: str
    patch_diff_sha256: str | None
    local_validation_report_sha256: str | None
    patch_application_manifest_ref: ArtifactReference | None
    post_patch_validation_report_ref: ArtifactReference | None

    @model_validator(mode="after")
    def _validate(self):
        if self.subject_type == "baseline":
            if self.variant_ids:
                raise ValueError("baseline workspace must have empty variant_ids")
            if self.patch_diff_sha256 is not None:
                raise ValueError("baseline workspace must have patch_diff_sha256=None")
            if self.local_validation_report_sha256 is not None:
                raise ValueError("baseline workspace must have local_validation_report_sha256=None")
            if self.patch_application_manifest_ref is not None:
                raise ValueError("baseline workspace must have patch_application_manifest_ref=None")
            if self.post_patch_validation_report_ref is not None:
                raise ValueError("baseline workspace must have post_patch_validation_report_ref=None")
        else:
            if not self.variant_ids:
                raise ValueError("variant workspace must have non-empty variant_ids")
            if self.patch_diff_sha256 is None:
                raise ValueError("variant workspace must have patch_diff_sha256")
            if self.local_validation_report_sha256 is None:
                raise ValueError("variant workspace must have local_validation_report_sha256")
            if self.patch_application_manifest_ref is None:
                raise ValueError("variant workspace must have patch_application_manifest_ref")
            if self.post_patch_validation_report_ref is None:
                raise ValueError("variant workspace must have post_patch_validation_report_ref")
        return self

class RunnerIntakeRequest(BaseModel):
    patch_runner_handoff_ref: ArtifactReference
    experiment_planner_handoff_sha256: str
    experiment_matrix_sha256: str
    shared_protocol_fingerprint: str
    statistical_analysis_plan_sha256: str
    operational_guard_policy_sha256: str
    workspace_refs: list[WorkspaceExecutionRef]

    @model_validator(mode="after")
    def _validate_workspaces(self):
        baseline_count = sum(1 for ws in self.workspace_refs if ws.subject_type == "baseline")
        if baseline_count != 1:
            raise ValueError("exactly 1 baseline workspace required")
        workspace_ids = [ws.workspace_id for ws in self.workspace_refs]
        if len(workspace_ids) != len(set(workspace_ids)):
            raise ValueError("duplicate workspace_id in workspace_refs")
        all_variant_ids = [
            vid for ws in self.workspace_refs if ws.subject_type == "variant"
            for vid in ws.variant_ids
        ]
        if len(all_variant_ids) != len(set(all_variant_ids)):
            raise ValueError("variant appears in multiple workspaces")
        return self

class RunnerIntakeReport(BaseModel):
    status: Literal["eligible", "blocked", "needs_revalidation"]
    checks: list[IntakeCheck]
    report_sha256: str
```

### ResourceUsageReport

```python
class ResourceUsageReport(BaseModel):
    """三态 nullable：measured / partially_measured / not_available。

    每一态通过 model_validator 独立校验字段一致性。
    """
    variant_id: str
    measurement_kind: Literal["measured", "partially_measured", "not_available"]
    measurement_tool: str | None

    peak_gpu_memory_mb: float | None
    avg_gpu_memory_mb: float | None
    peak_gpu_utilization_pct: float | None
    avg_gpu_utilization_pct: float | None
    wall_time_seconds: float | None
    cpu_time_seconds: float | None
    peak_cpu_memory_mb: float | None

    evidence_refs: list[ArtifactReference]

    @model_validator(mode="after")
    def _validate_fields(self):
        fields = [
            self.peak_gpu_memory_mb, self.avg_gpu_memory_mb,
            self.peak_gpu_utilization_pct, self.avg_gpu_utilization_pct,
            self.wall_time_seconds, self.cpu_time_seconds,
            self.peak_cpu_memory_mb,
        ]
        filled = [f for f in fields if f is not None]
        if self.measurement_kind == "measured":
            if not self.measurement_tool:
                raise ValueError("measured requires measurement_tool")
            if len(filled) != len(fields):
                raise ValueError("measured requires all fields non-None")
        elif self.measurement_kind == "partially_measured":
            if not self.measurement_tool:
                raise ValueError("partially_measured requires measurement_tool")
            if len(filled) < 1:
                raise ValueError("partially_measured requires at least one field")
        elif self.measurement_kind == "not_available":
            if filled:
                raise ValueError("not_available requires all fields None")
            if not self.evidence_refs:
                raise ValueError("not_available requires evidence_refs")
        return self
```

### RetryDecision

```python
class FailureClassification(str, Enum):
    TRANSIENT = "transient"
    RESOURCE_EXHAUSTION = "resource_exhaustion"
    CODE_BUG = "code_bug"
    DESIGN_FLAW = "design_flaw"
    ENVIRONMENT_ISSUE = "environment_issue"
    DATA_ISSUE = "data_issue"
    UNKNOWN = "unknown"

class RetryIdentity(BaseModel):
    execution_unit_plan_sha256: str
    command_sha256: str
    input_refs_sha256: str
    workspace_repository_fingerprint: str

class RetryDecision(BaseModel):
    attempt_id: str
    unit_id: str
    can_retry: bool
    retry_kind: Literal["retry_same_command", "do_not_retry", "needs_replan"]
    failure_classification: FailureClassification
    reason: str
    identity_match: bool

    @model_validator(mode="after")
    def _validate(self):
        if self.can_retry:
            if self.retry_kind != "retry_same_command":
                raise ValueError("can_retry=True requires retry_same_command")
            if not self.identity_match:
                raise ValueError("can_retry requires identity_match=True")
            if self.failure_classification in (FailureClassification.CODE_BUG, FailureClassification.DESIGN_FLAW):
                raise ValueError(f"failure_classification={self.failure_classification} incompatible with retry")
        else:
            if self.retry_kind != "do_not_retry":
                raise ValueError("can_retry=False requires do_not_retry")
        return self
```

# 3. ExecutionUnitRecord + Manifest + Handoff

```python
class ExecutionUnitStatus(str, Enum):
    COMPLETED = "completed"
    FAILED = "failed"
    BLOCKED = "blocked"

class ExecutionUnitRecord(BaseModel):
    unit_id: str
    matrix_entry_id: str
    variant_id: str | None
    seed: int | None
    stage: str
    workspace_id: str
    final_status: ExecutionUnitStatus
    final_attempt_id: str | None
    attempts: list[AttemptRecord]

    @model_validator(mode="after")
    def _final_status_from_attempts(self):
        if not self.attempts:
            raise ValueError("ExecutionUnitRecord requires at least one attempt")
        last = self.attempts[-1]
        if self.final_attempt_id is None:
            raise ValueError("final_attempt_id must be set when attempts exist")
        if self.final_attempt_id != last.attempt_id:
            raise ValueError("final_attempt_id must match last attempt")
        # final_status 由 last attempt 的结果派生
        return self

class MatrixCoverageReport(BaseModel):
    total_matrix_entries: int
    executable_entries: int
    executed_entries: int
    skipped_entry_ids: list[str]
    missing_unit_ids: list[str]

class ExecutionManifest(BaseModel):
    run_id: str
    experiment_matrix_sha256: str
    protocol_fingerprint: str
    workspace_refs_sha256: str
    operational_guard_policy_sha256: str
    runner_intake_report_ref: ArtifactReference
    unit_records: list[ExecutionUnitRecord]
    completed_unit_count: int
    failed_unit_count: int
    blocked_unit_count: int
    retry_decisions: list[RetryDecision]
    matrix_coverage: MatrixCoverageReport

class ExperimentExecutionHandoff(BaseModel):
    run_id: str
    execution_manifest_ref: ArtifactReference
    execution_unit_plans_sha256: str
    experiment_matrix_sha256: str
    statistical_analysis_plan_sha256: str
    protocol_fingerprint: str
    runner_intake_report_ref: ArtifactReference
    resource_budget_ref: ArtifactReference
    budget_decision_ref: ArtifactReference
    workspace_refs: list[WorkspaceExecutionRef]
    completed_unit_ids: list[str]
    failed_unit_ids: list[str]
    blocked_unit_ids: list[str]
    overall_status: Literal["completed", "partially_completed", "failed", "blocked"]
    next_stage: Literal["3.9_results_analysis"]

    @model_validator(mode="after")
    def _id_sets_match_manifest(self):
        completed = set(self.completed_unit_ids)
        failed = set(self.failed_unit_ids)
        blocked = set(self.blocked_unit_ids)
        if completed & failed or failed & blocked or completed & blocked:
            raise ValueError("unit ID sets must be disjoint")
        return self
```

# 4. 状态机与 Resume

## 4.1 状态机

```
unit.env → preflight → attempt_01 (run)
                    ↓
         success → ExecutionUnitRecord (COMPLETED)
         ↓
         failure → FailureClassification
             ├─ transient / resource → retry_same_command → attempt_02
             ├─ code_bug / design_flaw → do_not_retry → FAILED
             └─ env/data → do_not_retry → BLOCKED

attempt_02 必须通过 identity matching:
  - execution_unit_plan_sha256 一致
  - command_sha256 一致
  - input_refs_sha256 一致
  - workspace_repository_fingerprint 一致

interruption 恢复:
  - 已完成 unit → skip，标记 matrix_coverage.executed_entries
  - 正在执行 attempt_01 的 unit → 重新 preflight + 重跑
  - 正在执行 attempt_02 的 unit → 视为 attempt_01 失败，重新 analyze + retry
  - 从未开始的 unit → 按 plan 正常执行

attempt directory:
  - logs_{attempt_id}/stdout.log, stderr.log
  - Each attempt artifact 落盘后 SHA 计算
  - completed unit 下游只读，不可修改
```

## 4.2 Resume 恢复规则

```
1. 读取现有 ExecutionManifest（如果存在）
2. completed_unit_ids → skip（标记 executed_entries）
3. failed/blocked unit → 检查是否 attempt_02 已执行
   - 已执行 → 保持原状态
   - attempt_01 只失败 → 重新尝试 retry 判断
4. pending unit → 按 ExecutionUnitPlan 正常执行
5. 重新生成覆盖报告（completed + failed + resumed）
```

## 5.2 明确拒绝的模式

| 模式 | 拒绝原因 |
|------|---------|
| LLM 自动修改代码后直接重跑（AI-Scientist） | 修改代码必须重新经过 3.6–3.7 审批 |
| 失败后绕过审批跑新实验（autoresearch "NEVER STOP" 全自由修改） | AutoAD 的实验循环受 `OperationalGuardPolicy` 和 `approved_patch` 约束 |
| 使用 review ensemble 覆盖统计结论（AI-Scientist `perform_review.py`） | 统计结论由 3.5 的 `StatisticalAnalysisPlan` 确定性地决定，不交给 LLM 投票 |
| 把展示表格作为 canonical 数据源（SWE-Agent emoji table） | 结构化 JSON artifacts 是正式事实源，展示层是派生产物 |
| AI-Scientist 的 `coder_prompt` 让 LLM 直接写 `experiment.py` | AutoAD 的 patch 由 3.6 生成，经 3.7 审批后 `git apply`，不是 LLM 直接写文件 |
