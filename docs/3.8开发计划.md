# 0. 阶段定义

```
ExperimentPlannerHandoff (3.5) + PatchRunnerHandoff v2 (3.7)
    ↓
Runner Intake Validator → RunnerIntakeReport
    ↓
Matrix → Immutable ExecutionUnitPlans (plan only, no runtime state)
    ↓
每个 ExecutionUnit:
  Environment/Asset resolution
  → Immutable ExperimentCommandPlan (3.0)
  → Preflight (3.0)
  → Attempt 01:
      → AttemptIdentitySnapshot
      → ResourceUsageReport (nullable fields)
      → MetricsReport (3.0 ParsedMetric)
      → ScientificValidityReport (3.0)
      → 失败分类 → RetryDecision
  → Attempt 02 (仅 retry_same_command 且 identity 匹配)
    ↓
ExecutionUnitRecord + ExecutionManifest (runtime state, not in plan)
    ↓
ExperimentExecutionHandoff → 3.9
```

## 0.1 关键设计原则

- `ExecutionUnitPlan` 是**不可变的规划 artifact**，SHA 一经计算永不改变
- 运行时解析结果（resolved artifact bindings, produced artifacts）进入 `ExecutionUnitRecord`
- Retry 必须同时满足两个条件：失败分类允许 retry + identity 完全匹配
- 所有 model 可直接实现，不保留伪代码缩写

# 2. 3.8 核心模型

## 2.1 WorkspaceExecutionRef（带条件 Validator）

```python
class WorkspaceExecutionRef(BaseModel):
    workspace_id: str
    subject_type: Literal["baseline", "variant"]
    variant_ids: list[str]
    repository_fingerprint: str
    repository_commit: str
    patch_diff_sha256: str | None
    local_validation_report_sha256: str | None
    patch_runner_handoff_ref: ArtifactRef | None

    @model_validator(mode="after")
    def _validate_subject_consistency(self):
        if self.subject_type == "baseline":
            if self.variant_ids:
                raise ValueError("baseline: variant_ids must be empty")
            for f in ["patch_diff_sha256", "local_validation_report_sha256",
                       "patch_runner_handoff_ref"]:
                if getattr(self, f) is not None:
                    raise ValueError(f"baseline: {f} must be None")
        else:
            if not self.variant_ids:
                raise ValueError("variant: variant_ids must be non-empty")
            if self.patch_diff_sha256 is None:
                raise ValueError("variant: patch_diff_sha256 required")
            if self.local_validation_report_sha256 is None:
                raise ValueError("variant: local_validation_report_sha256 required")
            if self.patch_runner_handoff_ref is None:
                raise ValueError("variant: patch_runner_handoff_ref required")
        return self
```

整体 Validator（在 `RunnerIntakeRequest` 层）：
```
恰好一个 baseline workspace
workspace_id 不重复
每个 selected variant 恰好出现一次
variant 不跨 workspace 重复
```

## 2.5 ResourceUsageReport（nullable）

```python
class ResourceUsageReport(BaseModel):
    attempt_id: str
    unit_id: str
    measurement_status: Literal["measured", "partially_measured", "not_available"]

    wall_time_seconds: float | None
    cpu_time_seconds: float | None
    peak_rss_bytes: int | None
    peak_gpu_memory_bytes: int | None
    gpu_seconds: float | None

    measurement_tool: str | None
    evidence_refs: list[ArtifactRef]

    @model_validator(mode="after")
    def _validate_status_fields(self):
        fields = [self.wall_time_seconds, self.cpu_time_seconds,
                   self.peak_rss_bytes, self.peak_gpu_memory_bytes, self.gpu_seconds]
        if self.measurement_status == "measured":
            if self.wall_time_seconds is None:
                raise ValueError("measured requires wall_time_seconds")
            if self.measurement_tool is None:
                raise ValueError("measured requires measurement_tool")
        elif self.measurement_status == "partially_measured":
            if all(f is None for f in fields):
                raise ValueError("partially_measured requires at least one field")
        elif self.measurement_status == "not_available":
            if any(f is not None for f in fields):
                raise ValueError("not_available requires all fields None")
            if not self.evidence_refs:
                raise ValueError("not_available requires evidence_refs")
        return self
```

## 2.6 RetryDecision

```python
class RetryDecision(BaseModel):
    unit_id: str
    failed_attempt_id: str
    decision: Literal["retry_same_command", "do_not_retry",
                       "return_to_3_5", "return_to_3_6_3_7", "blocked"]
    reason_code: str
    evidence_refs: list[ArtifactRef]
```

**失败分类 → 是否允许 retry**：
```
瞬时错误/资源抢占 → retry_same_command
metric_parse_failed     → do_not_retry
validity invalid        → do_not_retry
invalid_repository_mutation → do_not_retry
代码/patch 错误         → return_to_3_6_3_7
实验设计缺陷            → return_to_3_5
```

**identity 匹配 → 确认是同一命令**：
```
if decision != "retry_same_command":
    return  # 不启动 attempt_02

if attempt_02.identity != attempt_01.identity:
    raise ValueError("retry with changed identity — new unit, not retry")

start_attempt_02()
```

identity 匹配是必要条件，不是充分条件。两者独立检查。

# 3. 状态机

```
intake_validating → environment_resolving → assets_resolving
  → command_binding → preflight_running → execution_running
  → metric_parsing → validity_checking
  → retry_decision_pending / unit_completed / unit_failed / unit_blocked

retry_decision_pending
  → retry_same_command + identity match → execution_running (attempt_02)
  → do_not_retry → unit_failed
  → return_to_3_5 / return_to_3_6_3_7 → unit_blocked

depends_on_unit 失败 → blocked_upstream_failure
```

Resume：
- unit_completed + artifact SHA 完整 → skip
- unit_running 无终态 → 标记 interrupted
- attempt 目录存在 → 不覆盖旧 evidence
- 上游 SHA 变化 → 全部 command plans 失效
