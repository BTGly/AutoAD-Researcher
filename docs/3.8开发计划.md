```

    ↓

    ↓
Matrix → ExecutionUnitPlans (精确 ArtifactBinding，seed 来自 Matrix)
    ↓
每个 ExecutionUnit:
  → ExperimentCommandPlan (3.0)
  → Preflight (3.0)
  → Attempt 01
  → AttemptIdentitySnapshot
  → ResourceUsageReport (nullable fields)
  → MetricsReport (3.0)
  → ScientificValidityReport (3.0)
  → RetryDecision (identity check via snapshot)
  → Attempt 02 (仅 retry_same_command)
    ↓
ExecutionManifest
    ↓
ExperimentExecutionHandoff → 3.9
```

Retry 身份检查：
```
attempt_02.identity.command_sha256 == attempt_01.identity.command_sha256
attempt_02.identity.input_refs_sha256 == attempt_01.identity.input_refs_sha256
attempt_02.identity.workspace_repository_fingerprint == attempt_01
→ retry_same_command
任一不匹配 → do_not_retry
```

## 2.4 ResourceUsageReport（nullable）

```python
class ResourceUsageReport(BaseModel):
    attempt_id: str; unit_id: str
    measurement_status: Literal["measured", "partially_measured", "not_available"]

    wall_time_seconds: float | None
    cpu_time_seconds: float | None
    peak_rss_bytes: int | None
    peak_gpu_memory_bytes: int | None
    gpu_seconds: float | None

    measurement_tool: str | None
    evidence_refs: list[ArtifactRef]

    @model_validator(mode="after")
    def _validate_status_fields(self):
        if self.measurement_status == "measured":
            if self.wall_time_seconds is None:
                raise ValueError("measured requires wall_time_seconds")
            if self.measurement_tool is None:
                raise ValueError("measured requires measurement_tool")
        elif self.measurement_status == "partially_measured":
            fields = [self.wall_time_seconds, self.cpu_time_seconds,
                       self.peak_rss_bytes, self.peak_gpu_memory_bytes, self.gpu_seconds]
            if all(f is None for f in fields):
                raise ValueError("partially_measured requires at least one field")
        elif self.measurement_status == "not_available":
            if any(f is not None for f in [self.wall_time_seconds, ...]):
                raise ValueError("not_available requires all resource fields to be None")
            if not self.evidence_refs:
                raise ValueError("not_available requires evidence_refs explaining why")
        return self
```
