# 0. 阶段定义

```
ExperimentPlannerHandoff (3.5) + PatchRunnerHandoff v2 (3.7, schema_version=2)
    ↓
Runner Intake Validator → RunnerIntakeReport
    ↓
Matrix → Immutable ExecutionUnitPlans
    ↓
每个 ExecutionUnit:
  → ExperimentCommandPlan (3.0) → Preflight (3.0)
  → Attempt 01: AttemptIdentitySnapshot → AttemptOutcome →
    ResourceUsageReport → MetricsReport → ScientificValidityReport → RetryDecision
  → Attempt 02 (仅 retry_same_command 且 identity 匹配)
    ↓
ExecutionUnitRecord + ExecutionManifest
    ↓
ExperimentExecutionHandoff → 3.9
```

## 0.1 关键原则

- `ExecutionUnitPlan` 不可变，SHA 一经计算永不改变
- 运行时解析结果入 `AttemptRecord`，不入 Plan
- 计划态 binding 有唯一 `binding_id`
- `RetryDecision` 以 `decision` 为唯一事实源（不含冗余 `can_retry` 布尔）
- `AttemptRecord` 必须携带确定性 `AttemptOutcome`（execution/metrics/validity 三态）
- `ExecutionUnitRecord.final_status` 由 `final_attempt.outcome` 派生，不是独立可写事实
- `ExperimentExecutionHandoff` 的 completed/failed/blocked 摘要必须与 Manifest 重算结果精确相等（service-level validator）
- 所有 schema Validator 使用 `raise ValueError`，禁止 `assert`
- 所有 artifact 引用使用 `ArtifactReferenceV2`（含 `sha256`），不使用 3.0 的 `ArtifactReference`

```python
            if self.patch_application_manifest_ref is None:
                raise ValueError("variant workspace must have patch_application_manifest_ref")

class RunnerIntakeRequest(BaseModel):
    patch_runner_handoff_ref: ArtifactReferenceV2
    experiment_planner_handoff_sha256: str
    experiment_matrix_sha256: str
    shared_protocol_fingerprint: str
    statistical_analysis_plan_sha256: str
    operational_guard_policy_sha256: str
    workspace_refs: list[WorkspaceExecutionRef]

    @model_validator(mode="after")
    def _validate_workspaces(self):
        baseline_count = sum(1 for ws in self.workspace_refs if ws.subject_type == "baseline")
        if baseline_count != 1:
            raise ValueError("exactly 1 baseline workspace required")
        workspace_ids = [ws.workspace_id for ws in self.workspace_refs]
        if len(workspace_ids) != len(set(workspace_ids)):
            raise ValueError("duplicate workspace_id in workspace_refs")
        all_variant_ids = [
            vid for ws in self.workspace_refs if ws.subject_type == "variant"
            for vid in ws.variant_ids
        ]
        if len(all_variant_ids) != len(set(all_variant_ids)):
            raise ValueError("variant appears in multiple workspaces")
        return self

class RunnerIntakeReport(BaseModel):

    report_sha256: str
```

### Intake ↔ PatchRunnerHandoff 交叉校验（P0-2，service layer）

`RunnerIntakeRequest` 不应同时保存 `patch_runner_handoff_ref` 和 `workspace_refs`（双事实源，可伪造）。更稳妥的设计是：Intake Validator 解引用 PatchRunnerHandoff 后自行生成 `workspace_refs`。

```python
def derive_workspace_execution_refs(
    handoff: PatchRunnerHandoff,
) -> list[WorkspaceExecutionRef]:
    """从 PatchRunnerHandoff v2 确定性派生 WorkspaceExecutionRef 列表。

    不信任调用者传入的 workspace_refs——由 Intake 层自行生成。
    """
    refs: list[WorkspaceExecutionRef] = []

    base = handoff.baseline_workspace_ref
    refs.append(WorkspaceExecutionRef(
        workspace_id=base.workspace_id,
        subject_type="baseline",
        variant_ids=[],
        repository_fingerprint=base.repository_fingerprint,
        repository_commit=base.repository_commit,
        patch_diff_sha256=None,
        local_validation_report_sha256=None,
        patch_application_manifest_ref=None,
        post_patch_validation_report_ref=None,
    ))

    for ws in handoff.variant_workspaces:
        refs.append(WorkspaceExecutionRef(
            workspace_id=ws.workspace_id,
            subject_type="variant",
            variant_ids=list(ws.variant_ids),
            repository_fingerprint=ws.repository_fingerprint,
            repository_commit=handoff.repository_before_commit,
            patch_diff_sha256=ws.patch_diff_sha256,
            local_validation_report_sha256=ws.local_validation_report_sha256,
            patch_application_manifest_ref=ws.patch_application_manifest_ref,
            post_patch_validation_report_ref=ws.post_patch_validation_report_ref,
        ))

    return refs

def validate_intake_against_patch_handoff(
    request: RunnerIntakeRequest,
    handoff: PatchRunnerHandoff,
) -> None:
    """Intake 层解引用 Handoff 后自行验证 workspace_refs 一致性。"""
    expected = derive_workspace_execution_refs(handoff)
    request_sha = canonical_sha256(request.workspace_refs)
    expected_sha = canonical_sha256(expected)

    if request_sha != expected_sha:
        raise ValueError("workspace_refs do not match PatchRunnerHandoff")
```

```python
class ResourceUsageReport(BaseModel):
    """资源报告粒度 = 一次 attempt，不是一个 variant。

    variant_id 在 baseline 时为 None，在 variant 时必填。
    seed 在 baseline/fit stage 可能为 None，在 full stage 必填。

    measurement_tool: str | None

    peak_gpu_memory_mb: float | None
    avg_gpu_memory_mb: float | None
    peak_gpu_utilization_pct: float | None
    avg_gpu_utilization_pct: float | None
    wall_time_seconds: float | None
    cpu_time_seconds: float | None
    peak_cpu_memory_mb: float | None

    evidence_refs: list[ArtifactReferenceV2]

    @model_validator(mode="after")
    def _validate(self):
        if self.subject_type == "baseline":
            if self.variant_id is not None:
                raise ValueError("baseline resource report must have variant_id=None")
        else:
            if self.variant_id is None:
                raise ValueError("variant resource report requires variant_id")

        fields = [
            self.peak_gpu_memory_mb, self.avg_gpu_memory_mb,
            self.peak_gpu_utilization_pct, self.avg_gpu_utilization_pct,
            self.wall_time_seconds, self.cpu_time_seconds,
            self.peak_cpu_memory_mb,
        ]
        filled = [f for f in fields if f is not None]
        if self.measurement_kind == "measured":
            if not self.measurement_tool:
                raise ValueError("measured requires measurement_tool")
            if len(filled) != len(fields):
                raise ValueError("measured requires all fields non-None")
        elif self.measurement_kind == "partially_measured":
            if not self.measurement_tool:
                raise ValueError("partially_measured requires measurement_tool")
            if len(filled) < 1:
                raise ValueError("partially_measured requires at least one field")
        elif self.measurement_kind == "not_available":
            if filled:
                raise ValueError("not_available requires all fields None")
            if not self.evidence_refs:
                raise ValueError("not_available requires evidence_refs")
        return self
```

```python
class FailureClassification(str, Enum):
    TRANSIENT = "transient"
    RESOURCE_EXHAUSTION = "resource_exhaustion"
    CODE_BUG = "code_bug"
    DESIGN_FLAW = "design_flaw"
    ENVIRONMENT_ISSUE = "environment_issue"
    DATA_ISSUE = "data_issue"
    UNKNOWN = "unknown"

class RetryIdentity(BaseModel):
    execution_unit_plan_sha256: str
    command_sha256: str
    input_refs_sha256: str
    workspace_repository_fingerprint: str

class RetryDecision(BaseModel):
    """以 decision 为唯一事实源。can_retry 布尔已移除（冗余且导致 needs_replan 不可达）。
    """
    attempt_id: str
    unit_id: str
    decision: Literal[
        "retry_same_command",
        "do_not_retry",
        "return_to_3_5",
        "return_to_3_6_3_7",
        "blocked",
    ]
    failure_classification: FailureClassification
    reason: str
    identity_match: bool

    @model_validator(mode="after")
    def _validate(self):
        if self.decision == "retry_same_command":
            if not self.identity_match:
                raise ValueError("retry_same_command requires identity_match=True")
            if self.failure_classification in (
                FailureClassification.CODE_BUG,
                FailureClassification.DESIGN_FLAW,
            ):
                raise ValueError(
                    f"failure_classification={self.failure_classification} "
                    "incompatible with retry_same_command"
                )
        return self
```

确定性映射（service-level，不是 schema validator）：

```
transient / resource_exhaustion
→ retry_same_command

code_bug / patch bug
→ return_to_3_6_3_7

design_flaw
→ return_to_3_5

invalid validity / deterministic parse failure
→ do_not_retry

unresolvable environment or data
→ blocked
```

`identity_match` 不应由调用者手工填写。应从两个 `AttemptIdentitySnapshot` 确定性计算：

```python
def compute_identity_match(
    prev_identity: AttemptIdentitySnapshot,
    next_identity: AttemptIdentitySnapshot,
) -> bool:
    return (
        prev_identity.execution_unit_plan_sha256 == next_identity.execution_unit_plan_sha256
        and prev_identity.command_sha256 == next_identity.command_sha256
        and prev_identity.input_refs_sha256 == next_identity.input_refs_sha256
        and prev_identity.workspace_repository_fingerprint == next_identity.workspace_repository_fingerprint
    )
```

# 4. ExecutionUnitRecord + Manifest + Handoff

```python
class ExecutionUnitStatus(str, Enum):
    COMPLETED = "completed"
    FAILED = "failed"
    BLOCKED = "blocked"

class ExecutionUnitRecord(BaseModel):
    unit_id: str
    matrix_entry_id: str
    variant_id: str | None
    seed: int | None
    stage: str
    workspace_id: str
    final_status: ExecutionUnitStatus
    final_attempt_id: str | None
    attempts: list[AttemptRecord]

    terminal_reason: Literal[
        "completed",
        "execution_failed",
        "validity_failed",
        "insufficient_evidence",
        "blocked_upstream_failure",
        "intake_failed",
        "preflight_failed",
    ]
    blocking_unit_ids: list[str]
    preflight_report_ref: ArtifactReferenceV2 | None

    @model_validator(mode="after")
    def _validate(self):
        if self.terminal_reason in ("completed", "execution_failed", "validity_failed", "insufficient_evidence"):
            if not self.attempts:
                raise ValueError("terminal_reason requires at least one attempt")
            if any(a.unit_id != self.unit_id for a in self.attempts):
                raise ValueError("attempt belongs to a different execution unit")
            if self.final_attempt_id is None:
                raise ValueError("final_attempt_id must be set")
            attempt_ids = [a.attempt_id for a in self.attempts]
            if len(attempt_ids) != len(set(attempt_ids)):
                raise ValueError("duplicate attempt_id in attempts")
            attempt_indices = [a.attempt_index for a in self.attempts]
            if attempt_indices[0] != 1:
                raise ValueError("attempt_index must start at 1")
            if attempt_indices != sorted(attempt_indices):
                raise ValueError("attempt_index must be strictly increasing")
            if len(attempt_indices) != len(set(attempt_indices)):
                raise ValueError("duplicate attempt_index in attempts")
            plan_shas = {a.identity.execution_unit_plan_sha256 for a in self.attempts}
            if len(plan_shas) != 1:
                raise ValueError("all attempts in a unit must share execution_unit_plan_sha256")
            last = self.attempts[-1]
            if self.final_attempt_id != last.attempt_id:
                raise ValueError("final_attempt_id must match last attempt")
            expected_reason = derive_terminal_reason(last.outcome)
            if self.terminal_reason != expected_reason:
                raise ValueError(
                    f"terminal_reason={self.terminal_reason} != "
                    f"derived={expected_reason} from outcome"
                )
            derived_status = derive_final_status(self.terminal_reason)
            if self.final_status != derived_status:
                raise ValueError(
                    f"final_status={self.final_status} != "
                    f"derived={derived_status} from terminal_reason={self.terminal_reason}"
                )
            if self.blocking_unit_ids:
                raise ValueError("attemptful terminal_reason must have empty blocking_unit_ids")
            if self.preflight_report_ref is not None:
                raise ValueError("attemptful terminal_reason must have preflight_report_ref=None")
        elif self.terminal_reason == "blocked_upstream_failure":
            if self.attempts:
                raise ValueError("blocked_upstream_failure requires zero attempts")
            if self.final_attempt_id is not None:
                raise ValueError("blocked_upstream_failure requires final_attempt_id=None")
            if not self.blocking_unit_ids:
                raise ValueError("blocked_upstream_failure requires blocking_unit_ids")
            if self.final_status != ExecutionUnitStatus.BLOCKED:
                raise ValueError("blocked_upstream_failure requires final_status=BLOCKED")
        elif self.terminal_reason == "preflight_failed":
            if self.attempts:
                raise ValueError("preflight_failed requires zero attempts")
            if self.final_attempt_id is not None:
                raise ValueError("preflight_failed requires final_attempt_id=None")
            if self.preflight_report_ref is None:
                raise ValueError("preflight_failed requires preflight_report_ref")
            if self.blocking_unit_ids:
                raise ValueError("preflight_failed must have empty blocking_unit_ids")
            if self.final_status != ExecutionUnitStatus.BLOCKED:
                raise ValueError("preflight_failed requires final_status=BLOCKED")
        elif self.terminal_reason == "intake_failed":
            if self.attempts:
                raise ValueError("intake_failed requires zero attempts")
            if self.final_attempt_id is not None:
                raise ValueError("intake_failed requires final_attempt_id=None")
            if self.preflight_report_ref is not None:
                raise ValueError("intake_failed must have preflight_report_ref=None")
            if self.blocking_unit_ids:
                raise ValueError("intake_failed must have empty blocking_unit_ids")
            if self.final_status != ExecutionUnitStatus.BLOCKED:
                raise ValueError("intake_failed requires final_status=BLOCKED")
        return self
```

### Service-level Handoff↔Manifest 一致性校验

schema validator 无法解引用 `execution_manifest_ref`。以下校验在 service layer 执行：

```python
def validate_handoff_against_manifest(
    handoff: ExperimentExecutionHandoff,
    manifest: ExecutionManifest,
) -> None:
    """Handoff 的所有摘要字段必须与 Manifest 精确相等。"""
    manifest_completed = {
        r.unit_id for r in manifest.unit_records
        if r.final_status == ExecutionUnitStatus.COMPLETED
    }
    manifest_failed = {
        r.unit_id for r in manifest.unit_records
        if r.final_status == ExecutionUnitStatus.FAILED
    }
    manifest_blocked = {
        r.unit_id for r in manifest.unit_records
        if r.final_status == ExecutionUnitStatus.BLOCKED
    }

    if set(handoff.completed_unit_ids) != manifest_completed:
        raise ValueError("handoff.completed_unit_ids != manifest derived set")
    if set(handoff.failed_unit_ids) != manifest_failed:
        raise ValueError("handoff.failed_unit_ids != manifest derived set")
    if set(handoff.blocked_unit_ids) != manifest_blocked:
        raise ValueError("handoff.blocked_unit_ids != manifest derived set")

    derived_overall = derive_overall_status(manifest)
    if handoff.overall_status != derived_overall:
        raise ValueError(
            f"handoff.overall_status={handoff.overall_status} != derived={derived_overall}"
        )

    if handoff.experiment_matrix_sha256 != manifest.experiment_matrix_sha256:
        raise ValueError("handoff.experiment_matrix_sha256 != manifest.experiment_matrix_sha256")
    if handoff.protocol_fingerprint != manifest.protocol_fingerprint:
        raise ValueError("handoff.protocol_fingerprint != manifest.protocol_fingerprint")
    if handoff.runner_intake_report_ref.sha256 != manifest.runner_intake_report_ref.sha256:
        raise ValueError("handoff.runner_intake_report_ref.sha256 != manifest.runner_intake_report_ref.sha256")

def derive_overall_status(manifest: ExecutionManifest) -> str:
    total = len(manifest.unit_records)
    completed = manifest.completed_unit_count
    failed = manifest.failed_unit_count
    blocked = manifest.blocked_unit_count
    if completed == total:
        return "completed"
    if blocked > 0 and failed == 0:
        return "blocked"
    if failed > 0 and completed > 0:
        return "partially_completed"
    if failed > 0 and completed == 0:
        return "failed"
    return "partially_completed"
```

# 5. AttemptOutcome ↔ Artifact 交叉校验（P0-3，service layer）

`AttemptRecord` 携带 `outcome: AttemptOutcome`，但 `outcome` 必须与引用的 artifacts 实际内容一致，不能仅仅自洽于 attempt record 内部字段。

```python
def validate_attempt_record_against_artifacts(
    attempt: AttemptRecord,
    execution_result: ExperimentExecutionResult | None,
    metrics_report: MetricsReport | None,
    validity_report: ScientificValidityReport | None,
    resource_report: ResourceUsageReport | None,
) -> None:
    """解引用 attempt 引用的 artifacts，验证 outcome 与实际内容一致。

    outcome 不是独立可写事实——它从 artifact 内容重算。
    """
    derived = derive_attempt_outcome(execution_result, metrics_report, validity_report)

    if attempt.outcome != derived:
        raise ValueError(
            f"AttemptOutcome does not match referenced artifacts: "
            f"stored={attempt.outcome}, derived={derived}"
        )

    if resource_report is not None:
        if resource_report.attempt_id != attempt.attempt_id:
            raise ValueError("resource_report.attempt_id != attempt.attempt_id")
        if resource_report.unit_id != attempt.unit_id:
            raise ValueError("resource_report.unit_id != attempt.unit_id")

    for binding in attempt.resolved_bindings:
        if binding.artifact_ref.sha256 != binding.artifact_sha256:
            raise ValueError("resolved_binding SHA mismatch")

    for prod in attempt.produced_artifacts:
        if prod.artifact_ref.sha256 != prod.artifact_sha256:
            raise ValueError("produced_artifact SHA mismatch")

def derive_execution_status(
    result: ExperimentExecutionResult | None,
) -> Literal["succeeded", "failed", "timeout", "not_run"]:
    """3.0 ExperimentExecutionResult.status 到 AttemptOutcome.execution_status 的显式适配。

    3.0 status 值: preflight_failed / execution_failed / metric_parse_failed /
                  invalid_repository_mutation / success
    3.0 timed_out: bool（独立字段）
    """
    if result is None:
        return "not_run"
    if result.timed_out:
        return "timeout"
    if result.status == "success":
        return "succeeded"
    return "failed"

def derive_attempt_outcome(
    execution_result: ExperimentExecutionResult | None,
    metrics_report: MetricsReport | None,
    validity_report: ScientificValidityReport | None,
) -> AttemptOutcome:
    """从 artifact 内容确定性派生 AttemptOutcome。"""
    if execution_result is None:
        return AttemptOutcome(
            execution_status="not_run",
            metrics_status="not_run",
            validity_status="not_run",
        )

    execution_status = derive_execution_status(execution_result)

    if metrics_report is None:
        metrics_status: Literal["parsed", "parse_failed", "not_run"] = "not_run"
    elif execution_result.status == "metric_parse_failed" or metrics_report.parse_errors:
        metrics_status = "parse_failed"
    else:
        metrics_status = "parsed"

    if validity_report is None:
        validity_status: Literal["valid", "invalid", "insufficient_evidence", "not_run"] = "not_run"
    else:
        validity_status = validity_report.overall_validity

    return AttemptOutcome(
        execution_status=execution_status,
        metrics_status=metrics_status,
        validity_status=validity_status,
    )


TerminalReason = Literal[
    "completed",
    "execution_failed",
    "validity_failed",
    "insufficient_evidence",
    "blocked_upstream_failure",
    "intake_failed",
    "preflight_failed",
]

def derive_terminal_reason(
    outcome: AttemptOutcome,
) -> TerminalReason:
    """从 AttemptOutcome 确定性派生 terminal_reason。

    terminal_reason → final_status 映射：
    completed               → COMPLETED
    execution_failed        → FAILED
    validity_failed         → FAILED
    insufficient_evidence   → BLOCKED
    blocked_upstream_failure→ BLOCKED
    intake_failed           → BLOCKED
    preflight_failed        → BLOCKED
    """
    if outcome.execution_status == "not_run":
        return "execution_failed"
    if outcome.execution_status == "timeout":
        return "execution_failed"
    if outcome.validity_status == "invalid":
        return "validity_failed"
    if outcome.validity_status == "insufficient_evidence":
        return "insufficient_evidence"
    if outcome.execution_status == "succeeded" and \
       outcome.metrics_status == "parsed" and \
       outcome.validity_status == "valid":
        return "completed"
    return "execution_failed"

_TERMINAL_REASON_TO_FINAL_STATUS: dict[TerminalReason, ExecutionUnitStatus] = {
    "completed": ExecutionUnitStatus.COMPLETED,
    "execution_failed": ExecutionUnitStatus.FAILED,
    "validity_failed": ExecutionUnitStatus.FAILED,
    "insufficient_evidence": ExecutionUnitStatus.BLOCKED,
    "blocked_upstream_failure": ExecutionUnitStatus.BLOCKED,
    "intake_failed": ExecutionUnitStatus.BLOCKED,
    "preflight_failed": ExecutionUnitStatus.BLOCKED,
}

def derive_final_status(reason: TerminalReason) -> ExecutionUnitStatus:
    return _TERMINAL_REASON_TO_FINAL_STATUS[reason]
```


# 7. 状态机与 Resume

## 7.1 状态机

```
unit.env → preflight → attempt_01 (run)
                    ↓
         outcome.execution=succeeded + metrics=parsed + validity=valid
           → ExecutionUnitRecord (COMPLETED)
         ↓
         outcome 不满足完成条件 → FailureClassification
             ├─ transient / resource_exhaustion → retry_same_command → attempt_02
             ├─ code_bug → return_to_3_6_3_7 → FAILED
             ├─ design_flaw → return_to_3_5 → FAILED
             ├─ invalid validity / parse_failed → do_not_retry → FAILED
             └─ env/data unresolvable → blocked → BLOCKED

attempt_02 必须通过 identity matching:
  - execution_unit_plan_sha256 一致
  - command_sha256 一致
  - input_refs_sha256 一致
  - workspace_repository_fingerprint 一致

interruption 恢复:
  - 已完成 unit → skip，标记 matrix_coverage.executed_entries
  - 正在执行 attempt_01 的 unit → 重新 preflight + 重跑
  - 正在执行 attempt_02 的 unit → 视为 attempt_01 失败，重新 analyze + retry
  - 从未开始的 unit → 按 plan 正常执行

attempt directory:
  - logs_{attempt_id}/stdout.log, stderr.log
  - Each attempt artifact 落盘后 SHA 计算
  - completed unit 下游只读，不可修改
```

## 7.2 Resume 恢复规则

```
1. 读取现有 ExecutionManifest（如果存在）
2. completed_unit_ids → skip（标记 executed_entries）
3. failed/blocked unit → 检查是否 attempt_02 已执行
   - 已执行 → 保持原状态
   - attempt_01 只失败 → 重新尝试 retry 判断
4. pending unit → 按 ExecutionUnitPlan 正常执行
5. 重新生成覆盖报告（completed + failed + resumed）
```
