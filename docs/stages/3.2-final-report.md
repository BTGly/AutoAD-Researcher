## Scope

Stage 3.2 seals the Unified Research Context Agent and Paper Intelligence
Capability for the current AutoAD-Researcher engineering baseline. The sealed
path is deterministic and CI-safe: it uses Pydantic v2 contracts, fixture-based
MinerU provider, canonical paper store, structured analysis/synthesis/validation/
repair loop, unified context assembly with gap classification and readiness
determination, and GitHub Actions verification.

This seal covers:

### Paper Intelligence Capability (P1-P9)

- **P1 Contracts** — PaperIntelligenceRequest, PaperAgentBudget, PaperClaim,
  PaperMentionedCandidate, PaperIdeaSourceCandidate, RepositoryLinkCandidate,
  MethodComponent, PaperSummary, PaperReaderResult, 5+1 evidence ref types,
  DocumentParseRequest/Result, ParserManifest, ParseQualityReport,
  PaperAnalysisControlSignal, validator

- **P2 MinerU Provider** — MinerUProvider Protocol, MinerUProfileConfig with
  locked profile SHA, FixtureMinerUProvider for offline testing,
  MINERU_PIPELINE_V1_PROFILE default

- **P3 Paper Tools** — CanonicalPaperStore (sections/blocks/pages),
  list_sections(), read_blocks(), search(), SectionInfo/PaperReadResult/
  PaperSearchMatch

- **P4 Skills** — paper-parse, paper-analysis, paper-synthesis, paper-repair
  SKILL.md files with frontmatter (required_tools, permission_profile, triggers)

- **P5 Profiles** — PAPER_PERMISSION_PROFILES (parse/analysis/synthesis/repair),

  PAPER_MODEL_ROUTING, PAPER_STAGE_SKILLS

## Key Design Decisions Preserved

1. physical_page_index is consistently 0-based
2. paper_fast_v1 includes tool_use capability
3. PaperAgentBudget includes max_analysis_llm_calls
4. PaperMentionedCandidate.selection_status locked to paper_mentioned
5. CandidateDecision is a separate artifact, never overwrites source candidates
6. Validator rejects unsupported claims (confirmed without evidence)
7. 3.3 is conditionally triggered (no-op when context is ready)
8. Reader evidence gaps never become user questions
9. capability status and readiness status are separately reported
