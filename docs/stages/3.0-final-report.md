## Scope

Stage 3.0 seals the engineering foundation needed before Step 3.1 Repository
Intelligence starts. It does not claim that the full 3.1 Tool Foundation exists
yet, and it does not claim that MVTec AD real benchmark evidence is complete.

This seal covers:

- environment planning, locking, validation, and revision contracts
- controlled asset preparation contracts
- immutable experiment command planning and execution evidence
- metrics parsing, scientific validity, and reproducibility report contracts
- offline fixtures, local verification, and GitHub Actions verification

This seal does not cover:

- generic `ToolSpec`, `ToolRegistry`, `ToolContext`, or `PermissionEngine`
- argv-based generic `ProcessTool`
- workspace-scoped generic Filesystem Tools
- Skill Loader
- Evidence Middleware
- State Attestor
- MVTec AD `attempt_01` / `attempt_02` real benchmark evidence
