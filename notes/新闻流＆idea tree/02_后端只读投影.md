# 02. 后端只读投影

## 1. 目标

新增一个前端专用的只读投影装配器，将 Session、IdeaTree、Attempt、OutcomeCard、CognitiveCommit、Champion 等权威状态组织为一次性返回的 `ExperimentProjection`。

## 2. 设计约束

- **不新增第二套权威状态** — 数据全部读自现有 Store
- **不修改实验 Agents 核心逻辑** — Coordinator、AttemptExecution、Finalizer、Promotion 不受影响
- **投影接口只读** — 不写入任何实验文件
- **事件不携带完整科研数据** — ActivityCard 由 event + event 引用的权威 artifact 组合生成

## 3. 新增文件

| 文件 | 职责 |
|------|------|
| `src/autoad_researcher/assistant/v2/experiment_projection.py` | 只读投影装配器 |
| `src/autoad_researcher/server/routes/experiment_projection.py` | API 路由 |
| `tests/test_v2_experiment_projection.py` | 投影装配器测试 |

## 4. 投影装配器接口

```python
# experiment_projection.py

@dataclass
class ExperimentProjection:
    schema_version: Literal[1] = 1
    
    # Session 基础
    session: SessionProjection | None
    session_list: list[SessionSummary]
    
    # 统计摘要
    summary: SessionStats
    
    # 科研状态
    idea_tree: IdeaTreeProjection | None
    attempts: list[AttemptProjection]
    cognitive_commits: list[CognitiveCommit]
    
    # Champion
    champion: ChampionInfo | None
    
    # 动态
    activity: list[ActivityCard]
    
    # 内部引用（仅开发者详情使用）
    developer_refs: DeveloperRefs
```

### 4.1 入口函数

```python
def build_projection(
    run_dir: Path,
    session_id: str | None = None,
) -> ExperimentProjection:
    """装配一个 Session 的只读投影。
    
    - session_id=None 且无 Session → 返回空投影（no_session=True）
    - session_id=None 且有多个 Session → 返回 session_list，不加载具体数据
    - session_id=None 且只有一个 Session → 自动选中
    - session_id 指定 → 加载该 Session
    """
```

### 4.10 ActivityCard

由 event + event 引用的权威 artifact 组合生成。使用确定性映射，不调用 LLM。

```python
@dataclass
class ActivityCard:
    event_id: int
    event_type: str
    created_at: str
    
    title: str  # 人类可读标题
    summary: str  # 1-2 行摘要
    card_kind: str  # 卡片类型：research_decision / implementation / experiment / result / failure / direction_change
    
    # 关联
    related_idea_id: str | None
    related_attempt_id: str | None
    related_commit_id: str | None
    related_outcome: dict | None  # OutcomeCard 的简化摘要
    
    # 详情
    detail: str  # 详细描述
    evidence_refs: list[str]
```

**事件 → 卡片映射表：**

| 事件类型 | card_kind | title 生成 |
|----------|-----------|------------|
| `experiment.session.created` | `session` | 实验 Session 已创建 |
| `experiment.idea_tree.created` | `decision` | Idea 树已初始化 |
| `experiment.idea_tree.mutated` (add_child) | `decision` | 新假设：`{读取新增节点的 mechanism}` |
| `experiment.idea_tree.mutated` (prune) | `direction_change` | 方向已停止：`{读取剪枝节点的 mechanism}` |
| `experiment.idea_tree.mutated` (mark_status → SUPPORTED) | `result` | 证据支持：`{读取节点的 mechanism}` |
| `experiment.idea_tree.mutated` (mark_status → NOT_SUPPORTED) | `result` | 未获支持：`{读取节点的 mechanism}` |
| `experiment.attempt.created` | `experiment` | 实验排队：`{purpose}` |
| `experiment.attempt.queued` | `experiment` | 实验加入队列 |
| `experiment.attempt.running` | `experiment` | 实验开始运行 |
| `experiment.attempt.finalized` | `result` | 实验完成：`{来源 OutcomeCard.scientific_effect}` |
| `experiment.attempt.retry_queued` | `failure` | 重试已排队 |
| `experiment.cognitive_commit.appended` | `decision` | 认知更新：`{来源 CognitiveCommit.hypothesis_verdict}` |
| `experiment.coordinator.exploratory_cycle.committed` | `decision` | 探索周期完成 |
| `experiment.coordinator.compact_cycle.committed` | `decision` | 收敛周期完成 |
| `experiment.coordinator.context_pruned` | `direction_change` | 上下文已压缩 |
| `experiment.champion.promoted_and_merged` | `result` | Champion 已更新：`{来源 CandidateSnapshot.idea_id}` |
| `experiment.champion.rolled_back` | `direction_change` | Champion 已回滚 |
| `experiment.stop_policy.evaluated` | `decision` | 停止策略评估完成 |
| `experiment.convergence.alert` | `direction_change` | 收敛告警：`{来源 alert 内容}` |

未知 event type → 研究者模式下忽略，开发者模式下保留原始 JSON。

**ActivityCard 装配器：**

```python
def _build_activity_card(
    event: dict,
    run_dir: Path,
    session_id: str,
    attempt_store: ExperimentAttemptStore,
    idea_tree: IdeaTree | None,
    commits: list[CognitiveCommit],
) -> ActivityCard | None:
    event_type = event["type"]
    payload = event.get("payload", {})
    
    if event_type == "experiment.attempt.finalized":
        attempt_id = payload.get("attempt_id", "")
        attempt = attempt_store.load(run_dir, attempt_id)
        outcome = _load_outcome_card(run_dir, attempt_id)
        if outcome:
            return ActivityCard(
                event_id=event["event_id"],
                event_type=event_type,
                created_at=event["created_at"],
                title=f"实验完成：{_effect_label(outcome.scientific_effect)}",
                summary=f"主指标变化：{_format_delta(outcome.primary_delta)}",
                card_kind="result",
                related_attempt_id=attempt_id,
                related_outcome=outcome.model_dump(mode="json"),
                # ...
            )
        return ActivityCard(
            event_id=event["event_id"],
            event_type=event_type,
            created_at=event["created_at"],
            title="实验已完成",
            summary="结果待读取",
            card_kind="result",
            related_attempt_id=attempt_id,
            # ...
        )
    
    if event_type == "experiment.idea_tree.mutated":
        return _build_idea_mutation_card(event, idea_tree)
    
    # ... 其余事件映射
```

## 5. API 路由

**新建文件：** `src/autoad_researcher/server/routes/experiment_projection.py`

```python
router = APIRouter()

@router.get("/api/runs/{run_id}/experiment/projection")
async def get_experiment_projection(
    run_id: str,
    session_id: str | None = Query(default=None),
):
    """返回一个 Session 的只读实验投影。"""
    run_dir = run_dir_path(RUNS_ROOT, run_id)
    projection = build_projection(run_dir, session_id=session_id)
    return projection
```

| 情况 | HTTP 状态 | body |
|------|-----------|------|
| run_id 不存在 | 404 | `{"error": "run not found"}` |
| session_id 不存在 | 404 | `{"error": "session not found"}` |
| session_id=None 且无 Session | 200 | `{"no_session": true}` |
| 正常 | 200 | `ExperimentProjection JSON` |
