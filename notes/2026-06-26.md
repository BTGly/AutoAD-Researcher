## Phase 1 UI: Local Streamlit Artifact Viewer + Preflight Runner

Added a local Streamlit dashboard for browsing L3 run artifacts and running
preflight. This is a **read-only operation console** ‚Äî no pipeline changes,
no approval enforcement.

### Files created

```
src/autoad_researcher/ui/
  __init__.py
  artifact_viewer.py     # read-only helpers for runs/{run_id}
  run_commands.py        # l3-preflight subprocess wrapper (no real L3)
  app.py                 # Streamlit 5-page dashboard
docs/l3/ui_runbook.md    # UI operation guide
```

### Modified

```
pyproject.toml           # added [project.optional-dependencies] ui = ["streamlit>=1.38"]
```

### 5 Pages

1. **Run Config** ‚Äî set run_id, dataset root, provider URL, mode, API key (password)
2. **Preflight Runner** ‚Äî one-click l3-preflight; real L3 only shows command text
3. **Artifact Explorer** ‚Äî per-stage file listing with sizes
4. **Execution Monitor** ‚Äî manifest, intake report, GPU evidence, events tail
5. **Final Review** ‚Äî three-panel status (Engineering ‚úÖ/‚ùå, GPU ‚úÖ/‚ùå, Scientific ‚úÖ/‚ùå)
   + artifact chain + final facts + final report markdown

### Post-review fixes (4 items)

1. **Preflight Runner safety** ‚Äî `run_preflight()` hardcodes `--mode l3-preflight`;
   the mode selectbox is disabled (`l3-preflight` only). Real L3 execution cannot be
   triggered from the UI button.
2. **Dataset root propagated** ‚Äî `run_preflight()` accepts `dataset_root` parameter,
   passed from UI input; written to subprocess env as `AUTOAD_INTERNAL_BENCHMARK_DATASET_ROOT`.
3. **run_dir_path validation** ‚Äî UI now delegates to `core.run_id.run_dir_path()`,
   rejecting empty, dot-only, and path-traversal run_ids. All 5 pages catch
   `ValueError` and display `st.error`.
4. **STAGE_NAMES complete** ‚Äî UI imports `STAGE3_ACCEPTANCE_STAGE_ORDER`
   (11 stages) + appends `stage3_acceptance` = 12 total.

### Key constraints

- API key stays in memory only ‚Äî never displayed, logged, or written to disk
- Real L3 execution NOT available from UI (command text only)
- No approval checkpoints ‚Äî annotation-only design deferred to Phase 2
- Entry point: `uv run --extra ui streamlit run src/autoad_researcher/ui/app.py`
