## 小型一致性修订 — 8 项设计修复 + 参考先入规则

### 操作

按用户反馈的 8 项设计一致性问题，对实验 Agents 设计包做统一修订：

1. **删除 CognitiveTaskRunner** → 直接复用 DeepAgents AgentFactory + SubAgentMiddleware (`01_大框架.md`)
2. **删除 StrategyOverlay/StrategyPolicy** → StrategySelector + DeepAgents SkillsMiddleware (`01_大框架.md`, `07_收敛.md`)
3. **Coordinator Compact Cycle 增加 PRUNED LESSONS** — 参考 Arbor `get_constraints_block()` 结构 (`03_Coordinator.md`)
4. **修复 Executor**: `git reset --hard HEAD~1` → `HEAD`，删除 `expected_activation_evidence`、`parameter_not_applied`、`hook_not_activated`，4 层 → 3 层 (`04_Executor.md`)
5. **PreApplyPatchGate 纯确定性化** — 删除语义偏移检测（loss/mechanism/training objective），只保留文件范围、protected SHA、worktree 边界、语法/import/smoke (`04_Executor.md`)
6. **PostRunFailureClassifier** — 替换 CognitiveTaskRunner 引用为 DeepAgents，修正 empty_transcript→LOST 判定规则（必须 PID/process group 确认），增加 6 层证据优先级 (`05_GPU监控.md`)
7. **ConvergenceMonitor → Arbor 滑动窗口** — `window_size=5, warn/paradigm_shift/stop` 可配置参数，替换旧 NORMAL/STAGNATING 等枚举 (`01_大框架.md`, `07_收敛.md`)
8. **PromotionTransaction → PromotionJournal `[COMPOSED]`** — = DVC experiment apply + Git merge refs + Optuna JournalStorage，修正 CandidateSnapshot 创建时序（B_dev 通过后立即创建，不等到 merge 后）(`01_大框架.md`, `06_有效性.md`)
