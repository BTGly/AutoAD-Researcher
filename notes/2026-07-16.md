# 2026-07-16

## Source classification and reuse boundary correction for experiment agents design plan

After cross-referencing the actual source code of all referenced projects (Arbor, SWE-Together,
software-agent-sdk, MiMo, AutoSOTA, AI-Scientist, claude-code-internals), identified that the
design plan used "直接复用" (direct reuse) inconsistently across multiple levels:
actual copy-paste, algorithm extraction, reimplementation from docs, and architecture reference.

### Changes made

- **00_README_文档索引.md**: Added 5-level reuse tag system (`[COPY]/[PORT]/[ADAPT]/[REIMPL]/[REFER]`)
  and source matrix recording repo, commit, path, license and reuse level for each external dependency.
- **01_实验Agents大框架.md**: Updated all reference tables with reuse level column; replaced
  "直接复用" wording with graded classification; renamed Sentinel → RuntimeWatchdog +
  PostRunFailureClassifier in architecture diagram.
- **04_ExecutorAgent与代码修改闭环.md**: Added §7 tool invocation spec (thin ToolSpec +
  3-stage pipeline + dual-stream context), explicitly scoped to avoid building a full framework.
- **05_ExperimentJob_GPU资源与训练监控.md**: Full rewrite of §5: split merged Sentinel into
  RuntimeWatchdog + PostRunFailureClassifier + AttemptFinalizer with clear progression;
  added DetectorProfile and FailureClassifierConfig with GPU_TRAINING profile;
  default disabled coding-agent-specific detectors.
- **06_实验有效性_Reflection与决策.md**: AutoSOTA reference corrected from "直接复用" to `[REIMPL]`;
  only implement 4 SHA guard functions, no record_score.sh pipeline logic.
- **07_收敛_认知预算与端到端验收.md**: MiMo `[PORT-PENDING-LICENSE]`; OpenHands 5-mode
  StuckDetector `[ADAPT-LATER]`; this round only implements cheap detection
  (repeated tool/patch/command).

### Key corrections

1. SWE-Together Sentinel is a post-hoc classifier, not a runtime daemon → split into two components.
2. AutoSOTA `record_score.sh` is proprietary pipeline code, not in open-source repo → `[REIMPL]`.
3. OpenHands StuckDetector is deeply coupled to OpenHands event type system → `[ADAPT-LATER]`.
4. GPU training vs coding-agent failure patterns must not mix → DetectorProfile isolation.
5. aider SEARCH/REPLACE strategy is ~100 lines of simple code → `[REIMPL]`, not AGPL copy.
